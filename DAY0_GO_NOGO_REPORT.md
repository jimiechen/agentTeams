## 9. Phase 1 建议

```
建议的 MVP 技术路径：
1. CDP 连接方式：chrome-remote-interface 直接连接 9222 端口
2. 文本注入方式：innerText + InputEvent('input', {bubbles: true})
3. 提交方式：document.querySelector('.chat-input-v2-send-button').click()
4. 响应检测方式：轮询 chat-turn 元素数量变化
5. 任务切换方式：document.querySelectorAll('.index-module__task-item___zOpfg')[index].click()

预估 MVP 工期：3-5 个工作日

需要特别关注：
1. DOM 选择器版本兼容性
2. AI 响应超时处理
3. 多任务并发控制

Day 0 遗留问题（可在 Phase 1 解决）：
1. 长时稳定性测试（更多循环）
2. 错误恢复机制
3. 状态持久化
```
