# 6.1 HeartbeatDaemon 的生命周期管理

## 类定义

```typescript
class HeartbeatDaemon {
  private running = false;
  private timers: { fast?: Timer; content?: Timer; deep?: Timer } = {};

  start(): void;      // 任务开始时调用
  stop(): void;       // 任务完成/中断时调用
  pause(): void;      // 进入 queuing 时降频
  resume(): void;     // 退出 queuing 后恢复
  isRunning(): boolean;
}
```

## 状态机

```
stopped ──start()──▶ running ──stop()──▶ stopped
running ──pause()──▶ paused  ──resume()──▶ running
running ──检测到 completed──▶ stopped（自动）
```

## 生命周期状态

| 状态 | 说明 | Timer 状态 |
|------|------|-----------|
| **stopped** | 守护进程未运行 | 所有 Timer 停止 |
| **running** | 守护进程正常运行 | 所有 Timer 按频率执行 |
| **paused** | 守护进程暂停（降频） | Timer 频率降低 |

## 状态转换

### start()
- **触发条件**：任务开始执行
- **动作**：启动所有 Timer，重置探测器和恢复执行器

### stop()
- **触发条件**：任务完成、任务中断、手动停止
- **动作**：停止所有 Timer，清理资源

### pause()
- **触发条件**：检测到 model-queuing
- **动作**：降低 Layer 1/2 的频率，保持 Layer 3 频率

### resume()
- **触发条件**：退出 model-queuing
- **动作**：恢复 Layer 1/2 的频率

## 状态检查

```typescript
isRunning(): boolean {
  return this.running;
}

isPaused(): boolean {
  return this.running && this.isPausedFlag;
}
```

## 生命周期钩子

### 启动钩子
```typescript
onStart(): void {
  // 可以在这里添加启动日志、初始化监控等
  console.log(`❤️ Heartbeat daemon started for ${this.workspace}`);
}
```

### 停止钩子
```typescript
onStop(): void {
  // 可以在这里添加停止日志、清理资源等
  console.log(`💔 Heartbeat daemon stopped for ${this.workspace}`);
}
```

## 并发保护

### 重复启动保护
```typescript
start(): void {
  if (this.running) {
    console.log('⚠️ Heartbeat daemon is already running');
    return;
  }
  // ...
}
```

### 重复停止保护
```typescript
stop(): void {
  if (!this.running) {
    console.log('⚠️ Heartbeat daemon is not running');
    return;
  }
  // ...
}
```

## 资源清理

### stop() 中的清理
```typescript
stop(): void {
  this.running = false;
  
  // 停止所有 Timer
  if (this.timers.fast) clearInterval(this.timers.fast);
  if (this.timers.content) clearInterval(this.timers.content);
  if (this.timers.deep) clearInterval(this.timers.deep);
  
  // 清理引用
  this.timers = {};
  
  // 调用停止钩子
  this.onStop();
}
```

### 优雅退出
确保在 Node 进程退出前停止所有守护进程：

```typescript
process.on('SIGINT', () => {
  // 停止所有心跳守护进程
  heartbeatDaemons.forEach(d => d.stop());
  process.exit(0);
});
```
