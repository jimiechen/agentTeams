# 6.2 三层 Timer 的启停策略

## 6.2.1 概述

HeartbeatDaemon 协调三层心跳 Timer（Layer 1/2/3），每层有不同的采集周期和职责。本节定义 Timer 的启动、暂停、停止策略，以及各层 Timer 之间的协调机制。

## 6.2.2 Timer 配置概览

| 层级 | 默认周期 | 职责 | 采集成本 |
|------|---------|------|---------|
| Layer 1 | 5 秒 | 快速状态检测，UI 元素采集 | ~3ms |
| Layer 2 | 15 秒 | 内容心跳，文本增量检测 | ~6.5ms |
| Layer 3 | 60 秒 | 深度心跳，健康度评估 | ~17.5ms |

## 6.2.5 启动策略

### 6.2.5.1 启动顺序

三层 Timer 按优先级顺序启动，确保 Layer 1 先于 Layer 2，Layer 2 先于 Layer 3：

```
启动时序:
T=0ms     Layer 1 启动
T=10ms    Layer 2 启动
T=20ms    Layer 3 启动
```

### 6.2.5.2 启动实现

```typescript
class TimerManager {
  async startAll(callbacks: TimerCallbacks): Promise<void> {
    const startOrder = ['layer1', 'layer2', 'layer3'] as const;

    for (const timerId of startOrder) {
      await this.startTimer(timerId, callbacks[timerId]);
      await this.sleep(10);
    }
  }

  private async startTimer(
    timerId: 'layer1' | 'layer2' | 'layer3',
    callback: () => Promise<void>
  ): Promise<void> {
    const timer = this._timers.get(timerId);
    if (!timer) return;

    if (timer.state === 'running') {
      console.warn(`Timer ${timerId} is already running`);
      return;
    }

    const interval = this.applyJitter(timer.interval);
    const now = Date.now();

    timer.handle = setInterval(async () => {
      await this.executeCallback(timerId, callback);
    }, interval);

    timer.state = 'running';
    timer.lastFireTime = now;
    timer.nextFireTime = now + interval;

    this.emit('timer:started', { timerId, interval });
  }

  private applyJitter(interval: number): number {
    if (!this._config.enableJitter) return interval;

    const jitter = interval * this._config.jitterRange;
    const offset = (Math.random() - 0.5) * 2 * jitter;
    return Math.max(1000, Math.round(interval + offset));
  }

  private async executeCallback(
    timerId: string,
    callback: () => Promise<void>
  ): Promise<void> {
    const timer = this._timers.get(timerId);
    if (!timer || timer.state !== 'running') return;

    const startTime = Date.now();
    timer.lastFireTime = startTime;

    try {
      await callback();
    } catch (error) {
      this.emit('timer:error', { timerId, error });
    }

    const executionTime = Date.now() - startTime;
    this.checkDrift(timerId, executionTime);
  }

  private checkDrift(timerId: string, executionTime: number): void {
    const timer = this._timers.get(timerId);
    if (!timer) return;

    if (executionTime > this._config.maxDrift) {
      this.emit('timer:drift', { timerId, drift: executionTime });
    }
  }

  private sleep(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}

interface TimerCallbacks {
  layer1: () => Promise<void>;
  layer2: () => Promise<void>;
  layer3: () => Promise<void>;
}
```

### 6.2.5.3 对齐启动策略

为避免三层 Timer 同时触发，采用对齐启动策略：

```typescript
class TimerManager {
  startAligned(callbacks: TimerCallbacks, alignmentMs: number = 0): void {
    const now = Date.now();

    const layer1 = this._timers.get('layer1')!;
    const layer2 = this._timers.get('layer2')!;
    const layer3 = this._timers.get('layer3')!;

    const layer1Delay = this.calculateAlignmentDelay(
      now, layer1.interval, alignmentMs
    );
    const layer2Delay = this.calculateAlignmentDelay(
      now, layer2.interval, alignmentMs + 1000
    );
    const layer3Delay = this.calculateAlignmentDelay(
      now, layer3.interval, alignmentMs + 2000
    );

    this.scheduleTimer('layer1', callbacks.layer1, layer1Delay);
    this.scheduleTimer('layer2', callbacks.layer2, layer2Delay);
    this.scheduleTimer('layer3', callbacks.layer3, layer3Delay);
  }

  private calculateAlignmentDelay(
    now: number,
    interval: number,
    offset: number
  ): number {
    const nextInterval = Math.ceil((now + offset) / interval) * interval;
    return nextInterval - now;
  }

  private scheduleTimer(
    timerId: string,
    callback: () => Promise<void>,
    delay: number
  ): void {
    const timer = this._timers.get(timerId);
    if (!timer) return;

    setTimeout(() => {
      this.startTimer(timerId as any, callback);
    }, delay);
  }
}
```

## 6.2.6 暂停策略

### 6.2.6.1 暂停机制

暂停时保留 Timer 的剩余时间，以便恢复时继续：

```typescript
class TimerManager {
  pauseAll(): void {
    const pauseOrder = ['layer3', 'layer2', 'layer1'] as const;

    for (const timerId of pauseOrder) {
      this.pauseTimer(timerId);
    }
  }

  private pauseTimer(timerId: string): void {
    const timer = this._timers.get(timerId);
    if (!timer || timer.state !== 'running') return;

    const now = Date.now();
    const remaining = timer.nextFireTime ? timer.nextFireTime - now : 0;

    this._pausedStates.set(timerId, Math.max(0, remaining));

    if (timer.handle) {
      clearInterval(timer.handle);
      timer.handle = null;
    }

    timer.state = 'paused';
    timer.remainingMs = remaining;

    this.emit('timer:paused', { timerId, remaining });
  }
}
```

### 6.2.6.2 暂停时机

| 场景 | 暂停策略 | 说明 |
|------|---------|------|
| 用户切换 Tab | 立即暂停 | 减少后台资源消耗 |
| 任务完成 | 等待当前周期完成后暂停 | 确保数据完整性 |
| 恢复动作执行中 | 暂停 Layer 1/2，保留 Layer 3 | 避免重复检测 |
| 网络断开 | 立即暂停 | 避免无效采集 |

## 6.2.7 恢复策略

### 6.2.7.1 恢复机制

```typescript
class TimerManager {
  resumeAll(callbacks: TimerCallbacks): void {
    const resumeOrder = ['layer1', 'layer2', 'layer3'] as const;

    for (const timerId of resumeOrder) {
      this.resumeTimer(timerId, callbacks[timerId]);
    }
  }

  private resumeTimer(
    timerId: 'layer1' | 'layer2' | 'layer3',
    callback: () => Promise<void>
  ): void {
    const timer = this._timers.get(timerId);
    if (!timer || timer.state !== 'paused') return;

    const remaining = this._pausedStates.get(timerId) || 0;

    if (remaining > 0) {
      setTimeout(async () => {
        await this.executeCallback(timerId, callback);
        this.startInterval(timerId, callback);
      }, remaining);
    } else {
      this.startInterval(timerId, callback);
    }

    timer.state = 'running';
    this._pausedStates.delete(timerId);

    this.emit('timer:resumed', { timerId, remaining });
  }

  private startInterval(
    timerId: 'layer1' | 'layer2' | 'layer3',
    callback: () => Promise<void>
  ): void {
    const timer = this._timers.get(timerId);
    if (!timer) return;

    const interval = timer.interval;

    timer.handle = setInterval(async () => {
      await this.executeCallback(timerId, callback);
    }, interval);

    timer.nextFireTime = Date.now() + interval;
  }
}
```

### 6.2.7.2 恢复时机

| 场景 | 恢复策略 | 说明 |
|------|---------|------|
| 用户回到 Tab | 立即恢复 | 继续监控 |
| 恢复动作完成 | 立即恢复 | 继续心跳检测 |
| 网络恢复 | 延迟 1 秒后恢复 | 等待网络稳定 |

## 6.2.8 停止策略

### 6.2.8.1 停止机制

```typescript
class TimerManager {
  stopAll(): void {
    const stopOrder = ['layer3', 'layer2', 'layer1'] as const;

    for (const timerId of stopOrder) {
      this.stopTimer(timerId);
    }

    this._pausedStates.clear();
  }

  private stopTimer(timerId: string): void {
    const timer = this._timers.get(timerId);
    if (!timer) return;

    if (timer.handle) {
      clearInterval(timer.handle);
      timer.handle = null;
    }

    timer.state = 'stopped';
    timer.lastFireTime = null;
    timer.nextFireTime = null;
    timer.remainingMs = null;

    this.emit('timer:stopped', { timerId });
  }

  reset(): void {
    this.stopAll();
    this.initializeTimers();
  }
}
```

### 6.2.8.2 优雅停止

等待当前执行中的回调完成后再停止：

```typescript
class TimerManager {
  private _pendingCallbacks: Set<string> = new Set();
  private _stopRequested = false;

  async stopGracefully(timeout: number = 5000): Promise<void> {
    this._stopRequested = true;

    const startTime = Date.now();

    while (this._pendingCallbacks.size > 0) {
      if (Date.now() - startTime > timeout) {
        console.warn('Graceful stop timeout, forcing stop');
        break;
      }
      await this.sleep(100);
    }

    this.stopAll();
    this._stopRequested = false;
  }

  private async executeCallback(
    timerId: string,
    callback: () => Promise<void>
  ): Promise<void> {
    if (this._stopRequested) return;

    this._pendingCallbacks.add(timerId);

    try {
      await callback();
    } finally {
      this._pendingCallbacks.delete(timerId);
    }
  }
}
```

## 6.2.9 动态调整策略

### 6.2.9.1 频率调整

根据系统负载动态调整 Timer 频率：

```typescript
class TimerManager {
  private _adaptiveMode = false;
  private _loadFactor = 1.0;

  enableAdaptiveMode(): void {
    this._adaptiveMode = true;
  }

  disableAdaptiveMode(): void {
    this._adaptiveMode = false;
    this._loadFactor = 1.0;
  }

  adjustForLoad(loadFactor: number): void {
    if (!this._adaptiveMode) return;

    this._loadFactor = Math.max(0.5, Math.min(2.0, loadFactor));

    this.adjustTimerIntervals();
  }

  private adjustTimerIntervals(): void {
    const baseIntervals = {
      layer1: this._config.layer1Interval,
      layer2: this._config.layer2Interval,
      layer3: this._config.layer3Interval
    };

    for (const [timerId, timer] of this._timers) {
      if (timer.state === 'running') {
        const newInterval = Math.round(
          baseIntervals[timerId as keyof typeof baseIntervals] * this._loadFactor
        );
        this.updateInterval(timerId, newInterval);
      }
    }
  }

  private updateInterval(timerId: string, newInterval: number): void {
    const timer = this._timers.get(timerId);
    if (!timer || timer.state !== 'running') return;

    if (timer.handle) {
      clearInterval(timer.handle);
    }

    timer.interval = newInterval;
    timer.handle = setInterval(() => {}, newInterval);
    timer.nextFireTime = Date.now() + newInterval;

    this.emit('timer:adjusted', { timerId, newInterval });
  }
}
```

### 6.2.9.2 场景化调整

| 场景 | Layer 1 | Layer 2 | Layer 3 | 说明 |
|------|---------|---------|---------|------|
| 正常运行 | 5s | 15s | 60s | 默认配置 |
| model-queuing | 5s | 30s | 120s | 降低检测频率 |
| model-stalled | 2s | 5s | 30s | 提高检测频率 |
| 恢复验证中 | 2s | 5s | 60s | 快速验证 |
| 低电量模式 | 10s | 30s | 120s | 节能模式 |

```typescript
const SCENE_INTERVALS: Record<string, TimerIntervals> = {
  normal: { layer1: 5000, layer2: 15000, layer3: 60000 },
  queuing: { layer1: 5000, layer2: 30000, layer3: 120000 },
  stalled: { layer1: 2000, layer2: 5000, layer3: 30000 },
  verifying: { layer1: 2000, layer2: 5000, layer3: 60000 },
  powerSaving: { layer1: 10000, layer2: 30000, layer3: 120000 }
};

interface TimerIntervals {
  layer1: number;
  layer2: number;
  layer3: number;
}

class TimerManager {
  applyScene(scene: keyof typeof SCENE_INTERVALS): void {
    const intervals = SCENE_INTERVALS[scene];
    if (!intervals) return;

    for (const [timerId, interval] of Object.entries(intervals)) {
      const timer = this._timers.get(timerId);
      if (timer) {
        timer.interval = interval;
        if (timer.state === 'running') {
          this.updateInterval(timerId, interval);
        }
      }
    }

    this.emit('timer:scene_changed', { scene, intervals });
  }
}
```

## 6.2.10 协调机制

### 6.2.10.1 层间协调

确保三层 Timer 不会同时触发：

```typescript
class TimerCoordinator {
  private _lastFireTimes: Map<string, number> = new Map();
  private _minSeparation = 500;

  shouldFire(timerId: string): boolean {
    const now = Date.now();

    for (const [id, lastFire] of this._lastFireTimes) {
      if (id !== timerId && now - lastFire < this._minSeparation) {
        return false;
      }
    }

    return true;
  }

  recordFire(timerId: string): void {
    this._lastFireTimes.set(timerId, Date.now());
  }

  getNextAvailableSlot(timerId: string): number {
    const now = Date.now();
    let minDelay = 0;

    for (const [id, lastFire] of this._lastFireTimes) {
      if (id !== timerId) {
        const elapsed = now - lastFire;
        const remaining = this._minSeparation - elapsed;
        if (remaining > minDelay) {
          minDelay = remaining;
        }
      }
    }

    return minDelay;
  }
}
```

### 6.2.10.2 优先级调度

```typescript
class TimerPriorityScheduler {
  private _priorities: Map<string, number> = new Map([
    ['layer1', 3],
    ['layer2', 2],
    ['layer3', 1]
  ]);

  private _pendingExecutions: Set<string> = new Set();

  async schedule(
    timerId: string,
    callback: () => Promise<void>
  ): Promise<void> {
    this._pendingExecutions.add(timerId);

    while (!this.canExecute(timerId)) {
      await this.sleep(50);
    }

    try {
      await callback();
    } finally {
      this._pendingExecutions.delete(timerId);
    }
  }

  private canExecute(timerId: string): boolean {
    const myPriority = this._priorities.get(timerId) || 0;

    for (const pending of this._pendingExecutions) {
      if (pending !== timerId) {
        const theirPriority = this._priorities.get(pending) || 0;
        if (theirPriority > myPriority) {
          return false;
        }
      }
    }

    return true;
  }

  private sleep(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

## 6.2.11 监控与诊断

### 6.2.11.1 Timer 统计

```typescript
interface TimerStats {
  timerId: string;
  totalFires: number;
  missedFires: number;
  avgExecutionTime: number;
  maxExecutionTime: number;
  avgDrift: number;
  maxDrift: number;
}

class TimerMonitor {
  private _stats: Map<string, TimerStats> = new Map();

  recordFire(timerId: string, executionTime: number, drift: number): void {
    const stats = this._stats.get(timerId) || {
      timerId,
      totalFires: 0,
      missedFires: 0,
      avgExecutionTime: 0,
      maxExecutionTime: 0,
      avgDrift: 0,
      maxDrift: 0
    };

    stats.totalFires++;
    stats.avgExecutionTime = this.updateAverage(
      stats.avgExecutionTime,
      executionTime,
      stats.totalFires
    );
    stats.maxExecutionTime = Math.max(stats.maxExecutionTime, executionTime);
    stats.avgDrift = this.updateAverage(
      stats.avgDrift,
      drift,
      stats.totalFires
    );
    stats.maxDrift = Math.max(stats.maxDrift, drift);

    this._stats.set(timerId, stats);
  }

  recordMiss(timerId: string): void {
    const stats = this._stats.get(timerId);
    if (stats) {
      stats.missedFires++;
    }
  }

  private updateAverage(current: number, value: number, n: number): number {
    return current * (n - 1) / n + value / n;
  }

  getStats(timerId?: string): TimerStats | TimerStats[] {
    if (timerId) {
      return this._stats.get(timerId) || null;
    }
    return Array.from(this._stats.values());
  }
}
```

### 6.2.11.2 健康检查

```typescript
class TimerHealthChecker {
  check(timers: Map<string, TimerInfo>): TimerHealthReport {
    const issues: TimerIssue[] = [];

    for (const [id, timer] of timers) {
      if (timer.state === 'running') {
        const expectedNextFire = timer.lastFireTime! + timer.interval;
        const drift = Math.abs(Date.now() - expectedNextFire);

        if (drift > 1000) {
          issues.push({
            timerId: id,
            type: 'drift',
            severity: drift > 5000 ? 'high' : 'medium',
            message: `Timer ${id} drift: ${drift}ms`
          });
        }
      }
    }

    return {
      healthy: issues.length === 0,
      issues,
      timestamp: Date.now()
    };
  }
}

interface TimerHealthReport {
  healthy: boolean;
  issues: TimerIssue[];
  timestamp: number;
}

interface TimerIssue {
  timerId: string;
  type: 'drift' | 'stall' | 'error';
  severity: 'low' | 'medium' | 'high';
  message: string;
}
```
