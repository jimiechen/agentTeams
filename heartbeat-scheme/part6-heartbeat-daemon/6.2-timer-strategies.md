# 6.2 三层 Timer 的启停策略

## 启动策略

```typescript
start(): void {
  if (this.running) return;
  this.running = true;
  this.detector.reset();
  this.recovery.reset();

  // Layer 1: 5 秒
  this.timers.fast = setInterval(async () => {
    try {
      const signal = await this.detector.heartbeat(false);
      if (signal.mode === 'completed') {
        this.stop();
        return;
      }
      if (signal.mode !== 'healthy') {
        const result = await this.recovery.recover(signal);
        this.logHeartbeat(signal, result.action);
      }
    } catch (err) {
      this.logError('layer1', err);
    }
  }, 5000);

  // Layer 2: 15 秒（offset 2.5s 避免与 Layer 1 同时触发）
  setTimeout(() => {
    this.timers.content = setInterval(async () => {
      try {
        await this.detector.contentCheck();
      } catch (err) {
        this.logError('layer2', err);
      }
    }, 15000);
  }, 2500);

  // Layer 3: 60 秒
  this.timers.deep = setInterval(async () => {
    try {
      const signal = await this.detector.heartbeat(true);
      this.logHeartbeat(signal, 'deep-snapshot');
    } catch (err) {
      this.logError('layer3', err);
    }
  }, 60000);
}
```

## offset 设计

### Layer 2 延迟启动
Layer 2 延迟 2.5 秒启动，避免和 Layer 1 在同一个事件循环 tick 触发，减少 CDP 调用冲突。

```
时间轴（秒）: 0    2.5     5     10    12.5    15    20    ...
Layer 1:      ●         ●      ●      ●        ●      ●      ●
Layer 2:              ●                ●               ●
Layer 3:                                              ●
```

## 停止策略

```typescript
stop(): void {
  this.running = false;
  if (this.timers.fast) clearInterval(this.timers.fast);
  if (this.timers.content) clearInterval(this.timers.content);
  if (this.timers.deep) clearInterval(this.timers.deep);
  this.timers = {};
}
```

## 暂停策略（降频）

```typescript
pause(): void {
  if (!this.running) return;
  
  // Layer 1: 从 5 秒降到 15 秒
  if (this.timers.fast) {
    clearInterval(this.timers.fast);
    this.timers.fast = setInterval(() => { /* Layer 1 逻辑 */ }, 15000);
  }
  
  // Layer 2: 从 15 秒降到 30 秒
  if (this.timers.content) {
    clearInterval(this.timers.content);
    this.timers.content = setInterval(() => { /* Layer 2 逻辑 */ }, 30000);
  }
  
  // Layer 3: 保持 60 秒不变
}
```

## 恢复策略

```typescript
resume(): void {
  if (!this.running) return;
  
  // Layer 1: 恢复到 5 秒
  if (this.timers.fast) {
    clearInterval(this.timers.fast);
    this.timers.fast = setInterval(() => { /* Layer 1 逻辑 */ }, 5000);
  }
  
  // Layer 2: 恢复到 15 秒
  if (this.timers.content) {
    clearInterval(this.timers.content);
    this.timers.content = setInterval(() => { /* Layer 2 逻辑 */ }, 15000);
  }
}
```

## Timer 状态管理

### Timer 引用管理
```typescript
private timers: {
  fast?: ReturnType<typeof setInterval>;
  content?: ReturnType<typeof setInterval>;
  deep?: ReturnType<typeof setInterval>;
} = {};
```

### 空值检查
每次操作前检查 Timer 是否存在，避免空值错误。

## 异常处理

### 单个 Timer 异常
单个 Timer 执行失败不会影响其他 Timer：

```typescript
this.timers.fast = setInterval(async () => {
  try {
    // Layer 1 逻辑
  } catch (err) {
    this.logError('layer1', err);
    // 继续下一次执行，不中断 Timer
  }
}, 5000);
```

### CDP 连接失败
CDP 连接失败时记录错误，但不停止 Timer，等待自动重连。
