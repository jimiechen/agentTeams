# 6.5 metrics 持久化

## 持久化策略

每次心跳决策（不论是否触发恢复）都追加一行 JSON 到 `runs/heartbeat.jsonl`。

## 实现代码

```typescript
private logHeartbeat(signal: HeartbeatSignal, action: string): void {
  const entry = {
    ts: Date.now(),
    workspace: this.workspace,
    mode: signal.mode,
    action,
    signal: {
      taskStatus: signal.taskStatus,
      charsDelta: signal.charsDelta,
      stalledSeconds: signal.stalledSeconds,
      queuePosition: signal.queuePosition,
      // 深度指标只在 Layer 3 心跳存在
      totalChars: signal.totalResponseChars,
      estimatedTokens: signal.estimatedTokens,
    },
  };
  appendFileSync(this.metricsPath, JSON.stringify(entry) + '\n');
}
```

## 文件路径

`workspaces/{workspace}/runs/heartbeat.jsonl`（和 `metrics.jsonl` 并列）。

## 轮转策略

### 文件大小限制
文件超过 10MB 时自动按天切分：

```typescript
private checkAndRotate(): void {
  const stats = fs.statSync(this.metricsPath);
  if (stats.size > 10 * 1024 * 1024) { // 10MB
    const date = new Date().toISOString().split('T')[0];
    const newPath = this.metricsPath.replace('.jsonl', `.${date}.jsonl`);
    fs.renameSync(this.metricsPath, newPath);
  }
}
```

### 归档策略
旧文件 7 天后自动归档到 zip（复用 Layer 0 原始记录的归档机制）。

## 数据格式

### JSONL 格式

```json
{"ts":1714356000000,"workspace":"PMCLI","mode":"healthy","action":"deep-snapshot","signal":{"taskStatus":"in_progress","charsDelta":150,"stalledSeconds":0,"queuePosition":null,"totalChars":5000,"estimatedTokens":2000}}
{"ts":1714356005000,"workspace":"PMCLI","mode":"healthy","action":"none","signal":{"taskStatus":"in_progress","charsDelta":200,"stalledSeconds":0,"queuePosition":null,"totalChars":5200,"estimatedTokens":2080}}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `ts` | number | 时间戳（毫秒） |
| `workspace` | string | 工作区名称 |
| `mode` | string | 当前健康模式 |
| `action` | string | 执行的动作 |
| `signal` | object | 信号快照 |

### signal 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `taskStatus` | string | 任务状态 |
| `charsDelta` | number | 字符增量 |
| `stalledSeconds` | number | 停滞秒数 |
| `queuePosition` | number \| null | 排队位置 |
| `totalChars` | number | 累计字符数 |
| `estimatedTokens` | number | 估算 Token 数 |

## 查询用例

### 查所有 model-stalled 事件
```bash
cat heartbeat.jsonl | jq 'select(.mode == "model-stalled")'
```

### 查某工作区的趋势
```bash
jq '.signal.estimatedTokens' heartbeat.jsonl | gnuplot
```

### 查恢复成功率
```bash
# 统计恢复动作数量
cat heartbeat.jsonl | jq 'select(.action | startswith("click:"))' | wc -l

# 统计恢复后变为 healthy 的比例
```

## 性能考虑

### 写入方式
使用 `appendFileSync` 追加写入，避免锁竞争。

### 异步写入（可选优化）
```typescript
// 批量异步写入
private async writeBatch(entries: Array<object>): Promise<void> {
  const content = entries.map(e => JSON.stringify(e)).join('\n') + '\n';
  await fs.promises.appendFile(this.metricsPath, content);
}
```

### 写入频率
- Layer 1：每 5 秒写入一次（仅当模式非 healthy 时）
- Layer 2：不单独写入
- Layer 3：每 60 秒写入一次（深度快照）

## 数据保留

### 保留期限
默认保留 30 天的历史数据。

### 清理策略
每天检查并删除超过 30 天的旧文件。
