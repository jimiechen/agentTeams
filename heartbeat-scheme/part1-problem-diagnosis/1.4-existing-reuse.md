# 1.4 与现有脚本能力的复用关系

`BUTTON_SCRIPTS_README.md` 里的四个检测函数是心跳机制的**底层能力**，不需要重写：

```
现有脚本                            心跳机制中的角色
──────────────────────────────────────────────────────
checkTaskStatus()        ──→    Layer 1 采集项之一
checkTerminalTimeout()   ──→    Layer 1 采集项之一
checkModelResponse()     ──→    Layer 2 采集项之一
checkUserPrompt()        ──→    Layer 1 采集项之一
```

现有脚本是**独立使用**的（每次手动调用一个），心跳机制把它们**组合成分层采样管道**，并在上层增加模式判定和恢复逻辑。
