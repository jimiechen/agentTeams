# 2.1 Layer 1：快速心跳（5秒周期）

## 2.1.1 设计目标

Layer 1 是三层心跳架构中最快的一层，以 **5 秒** 为周期执行轻量级采集。其核心设计目标是：

1. **快速响应**：在用户可感知的延迟内（5-10秒）发现界面状态变化
2. **低开销**：单次采集成本控制在 5ms 以内，不影响 Trae 主进程性能
3. **高可靠性**：作为最频繁的心跳层，必须具备极高的稳定性
4. **状态感知**：准确捕获任务状态、按钮状态等关键 UI 元素

## 2.1.2 采集项清单

| 采集项 | 数据类型 | 采集方式 | 用途 |
|--------|----------|----------|------|
| `taskStatus` | `string \| null` | DOM 查询任务列表状态文本 | 检测任务中断/完成 |
| `hasBackgroundBtn` | `boolean` | 查询"后台运行"按钮存在性 | 检测 terminal-hang |
| `hasCancelBtn` | `boolean` | 查询"取消"按钮存在性 | 检测任务执行中 |
| `hasRetainDeleteBtns` | `boolean` | 查询"保留/删除"按钮存在性 | 检测 modal-blocking |
| `activeTaskId` | `string \| null` | 查询当前高亮任务 ID | 任务追踪 |
| `timestamp` | `number` | `Date.now()` | 时间戳对齐 |

## 2.1.3 数据类型定义

```typescript
interface Layer1Payload {
  timestamp: number;
  taskStatus: string | null;
  hasBackgroundBtn: boolean;
  hasCancelBtn: boolean;
  hasRetainDeleteBtns: boolean;
  activeTaskId: string | null;
}

interface Layer1Result {
  layer: 1;
  payload: Layer1Payload;
  mode: Layer1Mode;
  cost: number;
}

type Layer1Mode = 
  | 'normal'
  | 'task-interrupted'
  | 'terminal-hang'
  | 'modal-blocking'
  | 'task-completed';
```

## 2.1.4 模式判定逻辑

```typescript
function determineLayer1Mode(payload: Layer1Payload): Layer1Mode {
  if (payload.taskStatus?.includes('中断')) {
    return 'task-interrupted';
  }
  
  if (payload.hasRetainDeleteBtns) {
    return 'modal-blocking';
  }
  
  if (payload.hasBackgroundBtn) {
    return 'terminal-hang';
  }
  
  if (payload.taskStatus === '完成') {
    return 'task-completed';
  }
  
  return 'normal';
}
```

## 2.1.5 采集实现代码

```typescript
class Layer1Collector {
  private readonly SELECTORS = {
    taskItem: '.index-module__task-item___zOpfg',
    taskStatus: '.task-status-text',
    backgroundBtn: 'button:has-text("后台运行")',
    cancelBtn: 'button:has-text("取消")',
    retainBtn: 'button:has-text("保留")',
    deleteBtn: 'button:has-text("删除")',
    activeTask: '.index-module__task-item--active___xyz'
  };

  async collect(page: Page): Promise<Layer1Result> {
    const startTime = Date.now();
    
    const payload: Layer1Payload = {
      timestamp: startTime,
      taskStatus: await this.getTaskStatus(page),
      hasBackgroundBtn: await this.hasElement(page, this.SELECTORS.backgroundBtn),
      hasCancelBtn: await this.hasElement(page, this.SELECTORS.cancelBtn),
      hasRetainDeleteBtns: await this.hasRetainDeleteButtons(page),
      activeTaskId: await this.getActiveTaskId(page)
    };
    
    return {
      layer: 1,
      payload,
      mode: determineLayer1Mode(payload),
      cost: Date.now() - startTime
    };
  }

  private async getTaskStatus(page: Page): Promise<string | null> {
    return page.evaluate(() => {
      const items = document.querySelectorAll('.index-module__task-item___zOpfg');
      for (const item of items) {
        if (item.classList.contains('index-module__task-item--active___xyz')) {
          const statusEl = item.querySelector('.task-status-text');
          return statusEl?.textContent?.trim() || null;
        }
      }
      return null;
    });
  }

  private async hasElement(page: Page, selector: string): Promise<boolean> {
    try {
      const element = await page.$(selector);
      return element !== null;
    } catch {
      return false;
    }
  }

  private async hasRetainDeleteButtons(page: Page): Promise<boolean> {
    const hasRetain = await this.hasElement(page, this.SELECTORS.retainBtn);
    const hasDelete = await this.hasElement(page, this.SELECTORS.deleteBtn);
    return hasRetain && hasDelete;
  }

  private async getActiveTaskId(page: Page): Promise<string | null> {
    return page.evaluate(() => {
      const activeItem = document.querySelector('.index-module__task-item--active___xyz');
      if (!activeItem) return null;
      
      const idAttr = activeItem.getAttribute('data-task-id');
      if (idAttr) return idAttr;
      
      const idEl = activeItem.querySelector('[data-id]');
      return idEl?.getAttribute('data-id') || null;
    });
  }
}
```

## 2.1.6 成本估算

| 项目 | 预估耗时 | 说明 |
|------|----------|------|
| DOM 查询 - taskStatus | ~1ms | 单次 `querySelector` |
| DOM 查询 - 按钮检测 | ~1ms | 并行执行 4 个 `querySelector` |
| DOM 查询 - activeTaskId | ~0.5ms | 单次 `querySelector` |
| 序列化与传输 | ~0.5ms | JSON 序列化 + IPC |
| **总计** | **~3ms** | 远低于 5ms 目标 |

### 每小时成本

- 采集次数：`3600 / 5 = 720 次/小时`
- 总耗时：`720 × 3ms = 2.16 秒/小时`
- CPU 占用率：`2.16 / 3600 ≈ 0.06%`

## 2.1.7 异常处理策略

```typescript
class Layer1Collector {
  private maxRetries = 3;
  private retryDelay = 100;

  async collectWithRetry(page: Page): Promise<Layer1Result | null> {
    for (let attempt = 1; attempt <= this.maxRetries; attempt++) {
      try {
        return await this.collect(page);
      } catch (error) {
        console.warn(`[Layer1] 采集失败 (尝试 ${attempt}/${this.maxRetries}):`, error);
        
        if (attempt < this.maxRetries) {
          await this.sleep(this.retryDelay * attempt);
        }
      }
    }
    
    return {
      layer: 1,
      payload: this.getEmptyPayload(),
      mode: 'normal',
      cost: -1
    };
  }

  private getEmptyPayload(): Layer1Payload {
    return {
      timestamp: Date.now(),
      taskStatus: null,
      hasBackgroundBtn: false,
      hasCancelBtn: false,
      hasRetainDeleteBtns: false,
      activeTaskId: null
    };
  }

  private sleep(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

## 2.1.9 监控指标

```typescript
interface Layer1Metrics {
  collectCount: number;
  successCount: number;
  failureCount: number;
  avgCost: number;
  maxCost: number;
  modeDistribution: Record<Layer1Mode, number>;
}

class Layer1Monitor {
  private metrics: Layer1Metrics = {
    collectCount: 0,
    successCount: 0,
    failureCount: 0,
    avgCost: 0,
    maxCost: 0,
    modeDistribution: {
      'normal': 0,
      'task-interrupted': 0,
      'terminal-hang': 0,
      'modal-blocking': 0,
      'task-completed': 0
    }
  };

  record(result: Layer1Result): void {
    this.metrics.collectCount++;
    
    if (result.cost >= 0) {
      this.metrics.successCount++;
      this.metrics.modeDistribution[result.mode]++;
      this.updateAvgCost(result.cost);
      this.metrics.maxCost = Math.max(this.metrics.maxCost, result.cost);
    } else {
      this.metrics.failureCount++;
    }
  }

  private updateAvgCost(cost: number): void {
    const n = this.metrics.successCount;
    this.metrics.avgCost = this.metrics.avgCost * (n - 1) / n + cost / n;
  }

  getMetrics(): Layer1Metrics {
    return { ...this.metrics };
  }
}
```
