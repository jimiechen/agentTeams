# 2.1 Layer 1 快速心跳（5秒）

## 职责定位

**职责**：在 5 秒内识别所有"按钮类显式异常"（terminal-hang、modal-blocking、task-interrupted）。

**目标**：快速响应、低延迟、轻量级

## 采集项清单

单次 CDP evaluate 一次性返回所有信号：

| 信号名称 | 来源 DOM | 数据类型 | 用途 |
|---------|---------|---------|------|
| `taskStatus` | `.index-module__task-item___zOpfg` | string | 判断任务状态（完成/进行中/中断） |
| `hasBackgroundButton` | `button[aria-label*="后台"]` | boolean | 检测 terminal-hang |
| `hasCancelButton` | `button[aria-label="取消"]` | boolean | 检测终端超时选项 |
| `hasKeepButton` | `button[aria-label*="保留"]` | boolean | 检测 modal-blocking |
| `hasDeleteButton` | `button[aria-label*="删除"]` | boolean | 检测 modal-blocking |

## CDP 执行脚本

```javascript
// Layer 1 快速探测脚本
async function layer1FastProbe(cdp) {
    return await cdp.evaluate(`(() => {
        const result = {
            taskStatus: 'unknown',
            hasBackgroundButton: false,
            hasCancelButton: false,
            hasKeepButton: false,
            hasDeleteButton: false
        };
        
        // 任务状态
        const taskItems = document.querySelectorAll('.index-module__task-item___zOpfg');
        for (const item of taskItems) {
            if (item.textContent?.includes('PMCLI')) {
                const text = item.textContent || '';
                if (text.includes('完成')) result.taskStatus = 'completed';
                else if (text.includes('进行中')) result.taskStatus = 'running';
                else if (text.includes('中断')) result.taskStatus = 'interrupted';
                break;
            }
        }
        
        // 按钮检测
        const buttons = document.querySelectorAll('button');
        buttons.forEach(btn => {
            const ariaLabel = btn.getAttribute('aria-label') || '';
            if (ariaLabel.includes('后台')) result.hasBackgroundButton = true;
            if (ariaLabel === '取消') result.hasCancelButton = true;
            if (ariaLabel.includes('保留')) result.hasKeepButton = true;
            if (ariaLabel.includes('删除')) result.hasDeleteButton = true;
        });
        
        return result;
    })()`);
}
```

## 成本估算

| 指标 | 数值 |
|------|------|
| 单次执行时间 | 50~100ms |
| CDP 调用次数 | 1 次 |
| DOM 查询数量 | ~30 个按钮 + 任务列表 |
| CPU 占比（5秒频率） | <1% |
| 网络开销 | ~1KB |

## 触发的模式判定

| 信号组合 | 判定模式 |
|---------|---------|
| `taskStatus === 'completed'` | completed |
| `taskStatus === 'interrupted'` | task-interrupted |
| `hasBackgroundButton === true` | terminal-hang |
| `hasKeepButton === true` 或 `hasDeleteButton === true` | modal-blocking |

## 状态更新策略

采集结果写入共享内存 `lastSignal.layer1`，供其他层级复用。

```javascript
// 共享内存结构
const lastSignal = {
    layer1: {
        timestamp: Date.now(),
        taskStatus: 'running',
        hasBackgroundButton: false,
        hasCancelButton: false,
        hasKeepButton: false,
        hasDeleteButton: false
    },
    layer2: { /* ... */ },
    layer3: { /* ... */ }
};
```

## 特殊处理

### 按钮去重
同一个按钮可能同时满足多个条件（如"保留并删除"按钮），需要按优先级只记录一个：

1. 后台运行按钮 > 取消按钮
2. 保留按钮 > 删除按钮

### 任务匹配
优先匹配包含"PMCLI"的任务项，如果不存在则取第一个任务项。
