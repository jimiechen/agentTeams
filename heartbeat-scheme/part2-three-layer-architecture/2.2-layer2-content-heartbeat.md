# 2.2 Layer 2 内容心跳（15秒）

## 职责定位

**职责**：识别 model-stalled 和 model-queuing——这两种模式的征兆都在"最后一条消息的文本内容"里。

**目标**：准确判定模型状态，避免误判

## 文本增量检测算法

### 核心原理

通过比较相邻两次采样的文本长度变化，判断模型是否停滞。

### 采样策略

```
每 15 秒采集 lastMessage.innerText.length
与上次采样对比，记录 charsDelta
```

### 停滞判定逻辑

```
条件 A：charsDelta == 0（本次采样与上次相比无增长）
条件 B：文本不含"排队"、"排在第 X 位"、"等待中"
条件 C：连续满足 A + B 两次（即 30 秒无增长）

三者同时满足 → model-stalled
```

### 排队识别

```javascript
// 排队位次提取
const queueMatch = text.match(/排在第\s*(\d+)\s*位/);
if (queueMatch) {
    queuePosition = parseInt(queueMatch[1]);
}
```

### 算法伪代码

```javascript
// Layer 2 内容检测算法
class ContentDetector {
    private lastCharCount = 0;
    private stalledCount = 0;
    private maxStalledCount = 2; // 2次 × 15秒 = 30秒
    
    detect(currentText) {
        const currentLength = currentText.length;
        const charsDelta = currentLength - this.lastCharCount;
        
        // 检查排队状态
        const isQueuing = currentText.includes('排队') || 
                         currentText.includes('排在第');
        
        if (isQueuing) {
            this.stalledCount = 0; // 排队不算停滞
            return { mode: 'model-queuing', queuePosition: this.extractQueuePosition(currentText) };
        }
        
        // 检查停滞
        if (charsDelta === 0) {
            this.stalledCount++;
            if (this.stalledCount >= this.maxStalledCount) {
                return { mode: 'model-stalled', stalledSeconds: this.stalledCount * 15 };
            }
        } else {
            this.stalledCount = 0; // 有增长，重置计数器
        }
        
        this.lastCharCount = currentLength;
        return { mode: 'healthy' };
    }
}
```

## 为什么是 15 秒

### 经验值依据

| 场景 | 典型时长 |
|------|---------|
| 模型"思考"阶段（调用工具、检索） | 5~10 秒 |
| 正常流式输出间隔 | < 2 秒 |
| 网络延迟 | < 1 秒 |

### 窗口选择理由

- **10 秒窗口**：容易误判（模型思考阶段可能暂停）
- **15 秒窗口**：合理平衡，配合 2 次连续判定共 30 秒
- **20 秒窗口**：响应太慢，用户体验差

### 连续判定的必要性

单次无增长可能是正常的"思考"阶段，连续两次无增长才说明真正停滞。

## CDP 执行脚本

```javascript
// Layer 2 内容探测脚本
async function layer2ContentProbe(cdp) {
    return await cdp.evaluate(`(() => {
        const turns = document.querySelectorAll('.chat-turn');
        const lastTurn = turns[turns.length - 1];
        
        if (!lastTurn) {
            return { text: '', isAiMessage: false };
        }
        
        // 判断是否为 AI 消息（非用户消息）
        const isUserMsg = lastTurn.classList.contains('user');
        
        return {
            text: lastTurn.innerText || '',
            isAiMessage: !isUserMsg,
            timestamp: Date.now()
        };
    })()`);
}
```

## 状态更新策略

采集结果写入共享内存 `lastSignal.layer2`：

```javascript
lastSignal.layer2 = {
    timestamp: Date.now(),
    textLength: currentLength,
    charsDelta: charsDelta,
    isQueuing: isQueuing,
    queuePosition: queuePosition,
    stalledSeconds: stalledSeconds
};
```

## 特殊处理

### 消息类型过滤
只分析 AI 消息的文本增长，用户消息不计入统计。

### 文本清洗
去除"复制图片"等干扰文本：

```javascript
text = text.replace(/复制图片/g, '').trim();
```

### 边界情况
- 无消息时：返回空文本，不计入判定
- 消息过短时：低于 50 字符不判定停滞（可能正在初始化）
