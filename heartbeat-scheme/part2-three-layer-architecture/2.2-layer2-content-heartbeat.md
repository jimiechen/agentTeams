# 2.2 Layer 2：内容心跳（15秒周期）

## 2.2.1 设计目标

Layer 2 是三层心跳架构的中间层，以 **15 秒** 为周期执行中等粒度的内容采集。其核心设计目标是：

1. **内容感知**：检测 AI 回复文本的增量变化，识别模型卡死（model-stalled）
2. **排队检测**：识别"排在第 X 位"等排队提示，区分正常等待与异常
3. **状态关联**：结合 Layer 1 的状态信息，进行更精准的模式判定
4. **适度开销**：单次采集成本控制在 10ms 以内

## 2.2.2 采集项清单

| 采集项 | 数据类型 | 采集方式 | 用途 |
|--------|----------|----------|------|
| `lastMessageContent` | `string \| null` | DOM 查询最后一条 AI 消息内容 | 文本增量计算 |
| `charsDelta` | `number` | 与上次采集的字符差值 | 检测 model-stalled |
| `queuePosition` | `number \| null` | 正则匹配"排在第 X 位" | 检测 model-queuing |
| `messageCount` | `number` | 查询消息总数 | 对话进度追踪 |
| `lastTurnIndex` | `number` | 最后一条消息的 turn 索引 | 消息定位 |
| `hasStreamingIndicator` | `boolean` | 查询流式输出指示器 | 检测输出中状态 |

## 2.2.3 数据类型定义

```typescript
interface Layer2Payload {
  timestamp: number;
  lastMessageContent: string | null;
  charsDelta: number;
  queuePosition: number | null;
  messageCount: number;
  lastTurnIndex: number;
  hasStreamingIndicator: boolean;
}

interface Layer2Result {
  layer: 2;
  payload: Layer2Payload;
  mode: Layer2Mode;
  cost: number;
  layer1Context?: Layer1Result;
}

type Layer2Mode =
  | 'normal'
  | 'model-stalled'
  | 'model-queuing'
  | 'model-streaming'
  | 'no-active-turn';

interface Layer2State {
  lastContent: string | null;
  lastTimestamp: number | null;
  consecutiveZeroDeltas: number;
}
```

## 2.2.4 模式判定逻辑

```typescript
const STALLED_THRESHOLD = 2;
const QUEUE_PATTERN = /排在第\s*(\d+)\s*位/;

function determineLayer2Mode(
  payload: Layer2Payload,
  state: Layer2State,
  layer1Context?: Layer1Result
): Layer2Mode {
  if (payload.lastMessageContent === null) {
    return 'no-active-turn';
  }
  
  if (payload.queuePosition !== null && payload.queuePosition > 0) {
    return 'model-queuing';
  }
  
  if (payload.hasStreamingIndicator) {
    return 'model-streaming';
  }
  
  if (layer1Context?.mode === 'terminal-hang' || 
      layer1Context?.mode === 'modal-blocking') {
    return 'normal';
  }
  
  if (state.consecutiveZeroDeltas >= STALLED_THRESHOLD) {
    return 'model-stalled';
  }
  
  return 'normal';
}
```

## 2.2.5 采集实现代码

```typescript
class Layer2Collector {
  private state: Layer2State = {
    lastContent: null,
    lastTimestamp: null,
    consecutiveZeroDeltas: 0
  };

  private readonly SELECTORS = {
    chatTurn: '.chat-turn',
    aiMessage: '.chat-turn.ai',
    markdownBody: '[class*="markdown-body"]',
    streamingIndicator: '.streaming-indicator, .typing-indicator'
  };

  async collect(page: Page, layer1Context?: Layer1Result): Promise<Layer2Result> {
    const startTime = Date.now();
    
    const lastMessageContent = await this.getLastMessageContent(page);
    const charsDelta = this.calculateCharsDelta(lastMessageContent);
    const queuePosition = this.extractQueuePosition(lastMessageContent);
    const { messageCount, lastTurnIndex } = await this.getMessageInfo(page);
    const hasStreamingIndicator = await this.hasStreamingIndicator(page);
    
    this.updateState(lastMessageContent, charsDelta);
    
    const payload: Layer2Payload = {
      timestamp: startTime,
      lastMessageContent: lastMessageContent?.slice(0, 500) || null,
      charsDelta,
      queuePosition,
      messageCount,
      lastTurnIndex,
      hasStreamingIndicator
    };
    
    return {
      layer: 2,
      payload,
      mode: determineLayer2Mode(payload, this.state, layer1Context),
      cost: Date.now() - startTime,
      layer1Context
    };
  }

  private async getLastMessageContent(page: Page): Promise<string | null> {
    return page.evaluate(() => {
      const turns = document.querySelectorAll('.chat-turn');
      if (turns.length === 0) return null;
      
      for (let i = turns.length - 1; i >= 0; i--) {
        const turn = turns[i];
        if (turn.classList.contains('ai') || 
            turn.querySelector('[class*="assistant"]')) {
          const body = turn.querySelector('[class*="markdown-body"]');
          return body?.innerText || null;
        }
      }
      return null;
    });
  }

  private calculateCharsDelta(currentContent: string | null): number {
    if (currentContent === null || this.state.lastContent === null) {
      return 0;
    }
    return currentContent.length - this.state.lastContent.length;
  }

  private extractQueuePosition(content: string | null): number | null {
    if (content === null) return null;
    
    const match = content.match(/排在第\s*(\d+)\s*位/);
    if (match) {
      return parseInt(match[1], 10);
    }
    return null;
  }

  private async getMessageInfo(page: Page): Promise<{
    messageCount: number;
    lastTurnIndex: number;
  }> {
    return page.evaluate(() => {
      const turns = document.querySelectorAll('.chat-turn');
      return {
        messageCount: turns.length,
        lastTurnIndex: turns.length - 1
      };
    });
  }

  private async hasStreamingIndicator(page: Page): Promise<boolean> {
    try {
      const indicator = await page.$(
        '.streaming-indicator, .typing-indicator, [data-streaming="true"]'
      );
      return indicator !== null;
    } catch {
      return false;
    }
  }

  private updateState(content: string | null, delta: number): void {
    this.state.lastContent = content;
    this.state.lastTimestamp = Date.now();
    
    if (delta === 0 && content !== null) {
      this.state.consecutiveZeroDeltas++;
    } else {
      this.state.consecutiveZeroDeltas = 0;
    }
  }

  resetState(): void {
    this.state = {
      lastContent: null,
      lastTimestamp: null,
      consecutiveZeroDeltas: 0
    };
  }

  getState(): Layer2State {
    return { ...this.state };
  }
}
```

## 2.2.6 成本估算

| 项目 | 预估耗时 | 说明 |
|------|----------|------|
| DOM 查询 - lastMessageContent | ~2ms | 遍历 chat-turn 列表 |
| 字符串处理 - charsDelta | ~0.5ms | 内存计算 |
| 正则匹配 - queuePosition | ~0.5ms | 简单正则 |
| DOM 查询 - messageInfo | ~1ms | `querySelectorAll` |
| DOM 查询 - streamingIndicator | ~1ms | 单次查询 |
| 状态更新与判定 | ~0.5ms | 内存操作 |
| 序列化与传输 | ~1ms | JSON 序列化 + IPC |
| **总计** | **~6.5ms** | 低于 10ms 目标 |

### 每小时成本

- 采集次数：`3600 / 15 = 240 次/小时`
- 总耗时：`240 × 6.5ms = 1.56 秒/小时`
- CPU 占用率：`1.56 / 3600 ≈ 0.04%`

## 2.2.7 model-stalled 检测机制

### 检测原理

`model-stalled` 是一种隐式异常，AI 模型没有返回错误，但输出停止增长。Layer 2 通过以下机制检测：

```
时间线:
T=0s    Layer2 采集: charsDelta = +50
T=15s   Layer2 采集: charsDelta = +30
T=30s   Layer2 采集: charsDelta = 0   ← consecutiveZeroDeltas = 1
T=45s   Layer2 采集: charsDelta = 0   ← consecutiveZeroDeltas = 2 → 触发 model-stalled
```

### 检测条件

```typescript
function isModelStalled(
  state: Layer2State,
  layer1Context?: Layer1Result
): boolean {
  if (state.consecutiveZeroDeltas < STALLED_THRESHOLD) {
    return false;
  }
  
  if (layer1Context?.mode === 'terminal-hang') {
    return false;
  }
  
  if (layer1Context?.mode === 'modal-blocking') {
    return false;
  }
  
  return true;
}
```

### 排除误判

以下情况不应判定为 `model-stalled`：

1. **terminal-hang**：终端命令阻塞，模型在等待
2. **modal-blocking**：对话框阻塞，需要用户操作
3. **model-queuing**：排队中，正常等待
4. **任务未开始**：没有活跃的 AI 消息

## 2.2.8 model-queuing 检测机制

### 排队提示识别

Trae 在请求量高时会显示排队提示，格式如：

```
请求量较高，排在第 5 位，请稍候...
当前排队中，您排在第 12 位
```

### 正则模式

```typescript
const QUEUE_PATTERNS = [
  /排在第\s*(\d+)\s*位/,
  /排队.*?(\d+)\s*位/,
  /等待.*?(\d+)\s*人/,
  /queue.*?(\d+)/i
];

function detectQueuePosition(content: string | null): number | null {
  if (content === null) return null;
  
  for (const pattern of QUEUE_PATTERNS) {
    const match = content.match(pattern);
    if (match) {
      return parseInt(match[1], 10);
    }
  }
  return null;
}
```

### 排队状态处理

```typescript
async function handleQueuing(
  result: Layer2Result,
  context: HeartbeatContext
): Promise<void> {
  const position = result.payload.queuePosition;
  
  if (position === null) return;
  
  if (position <= 3) {
    context.logger.info(`排队中，位置: ${position}，继续等待`);
  } else if (position <= 10) {
    context.logger.warn(`排队较长，位置: ${position}`);
    context.metrics.record('queue_wait', { position });
  } else {
    context.logger.error(`排队过长，位置: ${position}，建议用户干预`);
    context.alerts.send('queue_long', { position });
  }
}
```

## 2.2.9 与 Layer 1 的协作

```typescript
class Layer2WithLayer1Integration {
  private layer1Collector: Layer1Collector;
  private layer2Collector: Layer2Collector;
  private lastLayer1Result: Layer1Result | null = null;

  async collectWithIntegration(page: Page): Promise<Layer2Result> {
    const layer1Result = await this.layer1Collector.collect(page);
    this.lastLayer1Result = layer1Result;
    
    const layer2Result = await this.layer2Collector.collect(
      page,
      layer1Result
    );
    
    return layer2Result;
  }

  shouldSkipLayer2(layer1Result: Layer1Result): boolean {
    return [
      'task-interrupted',
      'modal-blocking'
    ].includes(layer1Result.mode);
  }

  getCombinedMode(layer1Result: Layer1Result, layer2Result: Layer2Result): string {
    if (layer1Result.mode !== 'normal') {
      return `layer1:${layer1Result.mode}`;
    }
    return `layer2:${layer2Result.mode}`;
  }
}
```

## 2.2.10 监控指标

```typescript
interface Layer2Metrics {
  collectCount: number;
  successCount: number;
  failureCount: number;
  avgCost: number;
  maxCost: number;
  modeDistribution: Record<Layer2Mode, number>;
  avgCharsDelta: number;
  stalledDetections: number;
  queueDetections: number;
  maxQueuePosition: number;
}

class Layer2Monitor {
  private metrics: Layer2Metrics = {
    collectCount: 0,
    successCount: 0,
    failureCount: 0,
    avgCost: 0,
    maxCost: 0,
    modeDistribution: {
      'normal': 0,
      'model-stalled': 0,
      'model-queuing': 0,
      'model-streaming': 0,
      'no-active-turn': 0
    },
    avgCharsDelta: 0,
    stalledDetections: 0,
    queueDetections: 0,
    maxQueuePosition: 0
  };

  record(result: Layer2Result): void {
    this.metrics.collectCount++;
    
    if (result.cost >= 0) {
      this.metrics.successCount++;
      this.metrics.modeDistribution[result.mode]++;
      this.updateAvgCost(result.cost);
      this.metrics.maxCost = Math.max(this.metrics.maxCost, result.cost);
      
      if (result.mode === 'model-stalled') {
        this.metrics.stalledDetections++;
      }
      
      if (result.mode === 'model-queuing') {
        this.metrics.queueDetections++;
        if (result.payload.queuePosition) {
          this.metrics.maxQueuePosition = Math.max(
            this.metrics.maxQueuePosition,
            result.payload.queuePosition
          );
        }
      }
    } else {
      this.metrics.failureCount++;
    }
  }

  private updateAvgCost(cost: number): void {
    const n = this.metrics.successCount;
    this.metrics.avgCost = this.metrics.avgCost * (n - 1) / n + cost / n;
  }

  getMetrics(): Layer2Metrics {
    return { ...this.metrics };
  }
}
```

## 2.2.13 状态重置场景

```typescript
class Layer2StateManager {
  private collector: Layer2Collector;

  shouldResetState(
    layer1Result: Layer1Result,
    layer2Result: Layer2Result
  ): boolean {
    if (layer1Result.mode === 'task-completed') {
      return true;
    }
    
    if (layer2Result.mode === 'model-stalled') {
      return true;
    }
    
    if (layer2Result.payload.charsDelta > 0) {
      const state = this.collector.getState();
      if (state.consecutiveZeroDeltas > 0) {
        return true;
      }
    }
    
    return false;
  }

  resetAfterRecovery(): void {
    this.collector.resetState();
    console.log('[Layer2] 状态已重置，开始新的监控周期');
  }
}
```
