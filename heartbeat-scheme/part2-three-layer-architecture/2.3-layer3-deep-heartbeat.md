# 2.3 Layer 3：深度心跳（60秒周期）

## 2.3.1 设计目标

Layer 3 是三层心跳架构中最深层、最全面的一层，以 **60 秒** 为周期执行全面的健康度采集。其核心设计目标是：

1. **健康度评估**：采集系统整体健康指标，评估 Trae 运行状态
2. **资源监控**：监控内存、CPU、网络等资源使用情况
3. **趋势分析**：收集历史数据，支持长期趋势分析和预测
4. **异常预警**：在问题爆发前发现潜在风险
5. **诊断支持**：为问题诊断提供详细的上下文信息

## 2.3.2 采集项清单

### 核心健康指标

| 采集项 | 数据类型 | 采集方式 | 用途 |
|--------|----------|----------|------|
| `memoryUsage` | `MemoryInfo` | `performance.memory` API | 内存泄漏检测 |
| `cpuUsage` | `number` | 估算 CPU 占用 | 性能监控 |
| `networkStatus` | `NetworkInfo` | `navigator.connection` | 网络状态 |
| `errorCount` | `number` | 全局错误计数器 | 错误追踪 |

### UI 状态指标

| 采集项 | 数据类型 | 采集方式 | 用途 |
|--------|----------|----------|------|
| `domNodeCount` | `number` | `document.querySelectorAll('*')` | DOM 复杂度 |
| `scrollPosition` | `ScrollInfo` | 滚动容器状态 | 滚动同步 |
| `visibleArea` | `VisibleArea` | 视口信息 | 可见性检测 |
| `inputFocus` | `boolean` | 输入框焦点状态 | 用户交互检测 |

### 任务状态指标

| 采集项 | 数据类型 | 采集方式 | 用途 |
|--------|----------|----------|------|
| `taskListSnapshot` | `TaskInfo[]` | 任务列表完整快照 | 任务进度追踪 |
| `conversationLength` | `number` | 对话消息总数 | 对话复杂度 |
| `fileOperationCount` | `number` | 文件操作计数 | 活跃度评估 |

### 性能指标

| 采集项 | 数据类型 | 采集方式 | 用途 |
|--------|----------|----------|------|
| `fps` | `number` | `requestAnimationFrame` 计算 | 渲染性能 |
| `responseTime` | `number` | 最近一次响应延迟 | 响应性能 |
| `layoutShifts` | `number` | Performance API | 布局稳定性 |

## 2.3.4 健康度评分算法

```typescript
const HEALTH_THRESHOLDS = {
  memory: {
    warning: 0.7,
    critical: 0.9
  },
  domNodes: {
    warning: 5000,
    critical: 10000
  },
  fps: {
    warning: 30,
    critical: 15
  },
  errorCount: {
    warning: 5,
    critical: 20
  },
  conversationLength: {
    warning: 100,
    critical: 200
  }
};

function calculateHealthScore(payload: Layer3Payload): {
  score: number;
  level: HealthLevel;
  factors: RiskFactor[];
} {
  let score = 100;
  const factors: RiskFactor[] = [];

  if (payload.memoryUsage) {
    const memUsage = payload.memoryUsage.usagePercentage;
    if (memUsage > HEALTH_THRESHOLDS.memory.critical) {
      score -= 30;
      factors.push({
        type: 'memory_high',
        severity: 'critical',
        message: `内存使用率过高: ${(memUsage * 100).toFixed(1)}%`,
        value: memUsage,
        threshold: HEALTH_THRESHOLDS.memory.critical
      });
    } else if (memUsage > HEALTH_THRESHOLDS.memory.warning) {
      score -= 15;
      factors.push({
        type: 'memory_high',
        severity: 'medium',
        message: `内存使用率偏高: ${(memUsage * 100).toFixed(1)}%`,
        value: memUsage,
        threshold: HEALTH_THRESHOLDS.memory.warning
      });
    }
  }

  if (payload.domNodeCount > HEALTH_THRESHOLDS.domNodes.critical) {
    score -= 25;
    factors.push({
      type: 'dom_bloat',
      severity: 'high',
      message: `DOM 节点过多: ${payload.domNodeCount}`,
      value: payload.domNodeCount,
      threshold: HEALTH_THRESHOLDS.domNodes.critical
    });
  } else if (payload.domNodeCount > HEALTH_THRESHOLDS.domNodes.warning) {
    score -= 10;
    factors.push({
      type: 'dom_bloat',
      severity: 'low',
      message: `DOM 节点偏多: ${payload.domNodeCount}`,
      value: payload.domNodeCount,
      threshold: HEALTH_THRESHOLDS.domNodes.warning
    });
  }

  if (payload.fps !== null && payload.fps < HEALTH_THRESHOLDS.fps.critical) {
    score -= 20;
    factors.push({
      type: 'fps_low',
      severity: 'high',
      message: `帧率过低: ${payload.fps} FPS`,
      value: payload.fps,
      threshold: HEALTH_THRESHOLDS.fps.critical
    });
  } else if (payload.fps !== null && payload.fps < HEALTH_THRESHOLDS.fps.warning) {
    score -= 10;
    factors.push({
      type: 'fps_low',
      severity: 'medium',
      message: `帧率偏低: ${payload.fps} FPS`,
      value: payload.fps,
      threshold: HEALTH_THRESHOLDS.fps.warning
    });
  }

  if (payload.errorCount > HEALTH_THRESHOLDS.errorCount.critical) {
    score -= 25;
    factors.push({
      type: 'error_spike',
      severity: 'critical',
      message: `错误数量过多: ${payload.errorCount}`,
      value: payload.errorCount,
      threshold: HEALTH_THRESHOLDS.errorCount.critical
    });
  } else if (payload.errorCount > HEALTH_THRESHOLDS.errorCount.warning) {
    score -= 10;
    factors.push({
      type: 'error_spike',
      severity: 'medium',
      message: `错误数量偏多: ${payload.errorCount}`,
      value: payload.errorCount,
      threshold: HEALTH_THRESHOLDS.errorCount.warning
    });
  }

  if (payload.conversationLength > HEALTH_THRESHOLDS.conversationLength.critical) {
    score -= 15;
    factors.push({
      type: 'conversation_long',
      severity: 'medium',
      message: `对话过长: ${payload.conversationLength} 条消息`,
      value: payload.conversationLength,
      threshold: HEALTH_THRESHOLDS.conversationLength.critical
    });
  }

  score = Math.max(0, score);

  let level: HealthLevel;
  if (score >= 80) level = 'healthy';
  else if (score >= 60) level = 'degraded';
  else if (score >= 40) level = 'unhealthy';
  else level = 'critical';

  return { score, level, factors };
}
```

## 2.3.5 采集实现代码

```typescript
class Layer3Collector {
  private errorCount = 0;
  private lastFrameTime = 0;
  private frameCount = 0;
  private currentFps: number | null = null;
  private fpsObserver: number | null = null;

  constructor() {
    this.setupErrorListener();
    this.startFpsMonitor();
  }

  private readonly SELECTORS = {
    taskItem: '.index-module__task-item___zOpfg',
    chatTurn: '.chat-turn',
    scrollContainer: '.chat-container, .messages-container',
    inputField: 'textarea, input[type="text"]'
  };

  async collect(
    page: Page,
    layer1Context?: Layer1Result,
    layer2Context?: Layer2Result
  ): Promise<Layer3Result> {
    const startTime = Date.now();

    const [
      memoryUsage,
      networkStatus,
      domNodeCount,
      scrollPosition,
      visibleArea,
      inputFocus,
      taskListSnapshot,
      conversationLength,
      layoutShifts
    ] = await Promise.all([
      this.getMemoryUsage(page),
      this.getNetworkStatus(page),
      this.getDomNodeCount(page),
      this.getScrollPosition(page),
      this.getVisibleArea(page),
      this.getInputFocus(page),
      this.getTaskListSnapshot(page),
      this.getConversationLength(page),
      this.getLayoutShifts(page)
    ]);

    const payload: Layer3Payload = {
      timestamp: startTime,
      memoryUsage,
      networkStatus,
      errorCount: this.errorCount,
      domNodeCount,
      scrollPosition,
      visibleArea,
      inputFocus,
      taskListSnapshot,
      conversationLength,
      fileOperationCount: 0,
      fps: this.currentFps,
      responseTime: null,
      layoutShifts
    };

    const { score, level, factors } = calculateHealthScore(payload);

    return {
      layer: 3,
      payload,
      healthScore: score,
      riskFactors: factors,
      cost: Date.now() - startTime,
      layer1Context,
      layer2Context
    };
  }

  private async getMemoryUsage(page: Page): Promise<MemoryInfo | null> {
    return page.evaluate(() => {
      const memory = (performance as any).memory;
      if (!memory) return null;

      return {
        usedJSHeapSize: memory.usedJSHeapSize,
        totalJSHeapSize: memory.totalJSHeapSize,
        jsHeapSizeLimit: memory.jsHeapSizeLimit,
        usagePercentage: memory.usedJSHeapSize / memory.jsHeapSizeLimit
      };
    });
  }

  private async getNetworkStatus(page: Page): Promise<NetworkInfo | null> {
    return page.evaluate(() => {
      const connection = (navigator as any).connection;
      if (!connection) {
        return {
          effectiveType: 'unknown',
          downlink: 0,
          rtt: 0,
          saveData: false,
          online: navigator.onLine
        };
      }

      return {
        effectiveType: connection.effectiveType || 'unknown',
        downlink: connection.downlink || 0,
        rtt: connection.rtt || 0,
        saveData: connection.saveData || false,
        online: navigator.onLine
      };
    });
  }

  private async getDomNodeCount(page: Page): Promise<number> {
    return page.evaluate(() => {
      return document.querySelectorAll('*').length;
    });
  }

  private async getScrollPosition(page: Page): Promise<ScrollInfo | null> {
    return page.evaluate(() => {
      const container = document.querySelector('.chat-container, .messages-container');
      if (!container) return null;

      return {
        scrollTop: container.scrollTop,
        scrollHeight: container.scrollHeight,
        clientHeight: container.clientHeight,
        scrollPercentage: container.scrollTop / (container.scrollHeight - container.clientHeight)
      };
    });
  }

  private async getVisibleArea(page: Page): Promise<VisibleArea | null> {
    return page.evaluate(() => {
      return {
        width: window.innerWidth,
        height: window.innerHeight,
        isMinimized: document.hidden,
        isVisible: !document.hidden && document.hasFocus()
      };
    });
  }

  private async getInputFocus(page: Page): Promise<boolean> {
    return page.evaluate(() => {
      const activeElement = document.activeElement;
      return activeElement?.tagName === 'TEXTAREA' || 
             activeElement?.tagName === 'INPUT';
    });
  }

  private async getTaskListSnapshot(page: Page): Promise<TaskInfo[]> {
    return page.evaluate(() => {
      const items = document.querySelectorAll('.index-module__task-item___zOpfg');
      const tasks: TaskInfo[] = [];

      items.forEach((item, index) => {
        const titleEl = item.querySelector('.task-title');
        const statusEl = item.querySelector('.task-status-text');
        
        tasks.push({
          id: item.getAttribute('data-task-id') || `task-${index}`,
          title: titleEl?.textContent?.trim() || '',
          status: statusEl?.textContent?.trim() || 'unknown',
          createdAt: Date.now(),
          updatedAt: Date.now()
        });
      });

      return tasks;
    });
  }

  private async getConversationLength(page: Page): Promise<number> {
    return page.evaluate(() => {
      return document.querySelectorAll('.chat-turn').length;
    });
  }

  private async getLayoutShifts(page: Page): Promise<number> {
    return page.evaluate(() => {
      let shifts = 0;
      const observer = new PerformanceObserver((list) => {
        for (const entry of list.getEntries()) {
          if (entry.entryType === 'layout-shift' && !(entry as any).hadRecentInput) {
            shifts++;
          }
        }
      });
      
      try {
        observer.observe({ type: 'layout-shift', buffered: true });
        const entries = performance.getEntriesByType('layout-shift');
        shifts = entries.filter((e: any) => !e.hadRecentInput).length;
      } catch {
        shifts = 0;
      }
      
      return shifts;
    });
  }

  private setupErrorListener(): void {
    if (typeof window !== 'undefined') {
      window.addEventListener('error', () => {
        this.errorCount++;
      });
      window.addEventListener('unhandledrejection', () => {
        this.errorCount++;
      });
    }
  }

  private startFpsMonitor(): void {
    if (typeof window === 'undefined') return;

    const measureFps = (timestamp: number) => {
      if (this.lastFrameTime === 0) {
        this.lastFrameTime = timestamp;
        this.frameCount = 0;
      } else {
        this.frameCount++;
        const elapsed = timestamp - this.lastFrameTime;
        if (elapsed >= 1000) {
          this.currentFps = Math.round((this.frameCount * 1000) / elapsed);
          this.lastFrameTime = timestamp;
          this.frameCount = 0;
        }
      }
      this.fpsObserver = requestAnimationFrame(measureFps);
    };

    this.fpsObserver = requestAnimationFrame(measureFps);
  }

  stopFpsMonitor(): void {
    if (this.fpsObserver !== null) {
      cancelAnimationFrame(this.fpsObserver);
      this.fpsObserver = null;
    }
  }

  resetErrorCount(): void {
    this.errorCount = 0;
  }
}
```

## 2.3.7 内存泄漏检测

```typescript
interface MemoryTrend {
  samples: number[];
  trend: 'increasing' | 'stable' | 'decreasing';
  leakProbability: number;
}

class MemoryLeakDetector {
  private samples: number[] = [];
  private readonly SAMPLE_SIZE = 10;
  private readonly LEAK_THRESHOLD = 0.05;

  addSample(usagePercentage: number): void {
    this.samples.push(usagePercentage);
    if (this.samples.length > this.SAMPLE_SIZE) {
      this.samples.shift();
    }
  }

  analyze(): MemoryTrend {
    if (this.samples.length < 3) {
      return {
        samples: this.samples,
        trend: 'stable',
        leakProbability: 0
      };
    }

    const increasing = this.countIncreasing();
    const total = this.samples.length - 1;
    const increaseRate = increasing / total;

    let trend: 'increasing' | 'stable' | 'decreasing';
    if (increaseRate > 0.7) trend = 'increasing';
    else if (increaseRate < 0.3) trend = 'decreasing';
    else trend = 'stable';

    const leakProbability = trend === 'increasing' ? 
      Math.min(1, increaseRate * 1.5) : 0;

    return {
      samples: [...this.samples],
      trend,
      leakProbability
    };
  }

  private countIncreasing(): number {
    let count = 0;
    for (let i = 1; i < this.samples.length; i++) {
      if (this.samples[i] > this.samples[i - 1]) {
        count++;
      }
    }
    return count;
  }

  isLeakDetected(): boolean {
    const analysis = this.analyze();
    return analysis.leakProbability > this.LEAK_THRESHOLD;
  }
}
```

## 2.3.8 与其他层的协作

```typescript
class Layer3Integration {
  private layer1Collector: Layer1Collector;
  private layer2Collector: Layer2Collector;
  private layer3Collector: Layer3Collector;

  async collectFullSnapshot(page: Page): Promise<{
    layer1: Layer1Result;
    layer2: Layer2Result;
    layer3: Layer3Result;
  }> {
    const layer1 = await this.layer1Collector.collect(page);
    const layer2 = await this.layer2Collector.collect(page, layer1);
    const layer3 = await this.layer3Collector.collect(page, layer1, layer2);

    return { layer1, layer2, layer3 };
  }

  shouldEscalateAlert(layer3: Layer3Result): boolean {
    return layer3.healthScore < 60 || 
           layer3.riskFactors.some(f => f.severity === 'critical');
  }

  getRecommendedAction(layer3: Layer3Result): string {
    const criticalFactors = layer3.riskFactors.filter(
      f => f.severity === 'critical'
    );

    if (criticalFactors.length === 0) {
      return 'continue';
    }

    const factor = criticalFactors[0];
    switch (factor.type) {
      case 'memory_high':
      case 'memory_leak':
        return 'restart_session';
      case 'dom_bloat':
        return 'clear_history';
      case 'fps_low':
        return 'reduce_load';
      case 'error_spike':
        return 'investigate_errors';
      default:
        return 'alert_user';
    }
  }
}
```

## 2.3.9 监控指标

```typescript
interface Layer3Metrics {
  collectCount: number;
  successCount: number;
  failureCount: number;
  avgCost: number;
  maxCost: number;
  avgHealthScore: number;
  minHealthScore: number;
  healthLevelDistribution: Record<HealthLevel, number>;
  riskFactorCounts: Record<RiskType, number>;
  avgMemoryUsage: number;
  avgFps: number;
  avgDomNodeCount: number;
}

class Layer3Monitor {
  private metrics: Layer3Metrics = {
    collectCount: 0,
    successCount: 0,
    failureCount: 0,
    avgCost: 0,
    maxCost: 0,
    avgHealthScore: 0,
    minHealthScore: 100,
    healthLevelDistribution: {
      'healthy': 0,
      'degraded': 0,
      'unhealthy': 0,
      'critical': 0
    },
    riskFactorCounts: {
      'memory_high': 0,
      'memory_leak': 0,
      'dom_bloat': 0,
      'fps_low': 0,
      'network_unstable': 0,
      'error_spike': 0,
      'conversation_long': 0,
      'response_slow': 0
    },
    avgMemoryUsage: 0,
    avgFps: 0,
    avgDomNodeCount: 0
  };

  record(result: Layer3Result): void {
    this.metrics.collectCount++;

    if (result.cost >= 0) {
      this.metrics.successCount++;
      this.updateAvgCost(result.cost);
      this.metrics.maxCost = Math.max(this.metrics.maxCost, result.cost);

      const level = this.getHealthLevel(result.healthScore);
      this.metrics.healthLevelDistribution[level]++;
      this.metrics.avgHealthScore = this.updateAverage(
        this.metrics.avgHealthScore,
        result.healthScore
      );
      this.metrics.minHealthScore = Math.min(
        this.metrics.minHealthScore,
        result.healthScore
      );

      for (const factor of result.riskFactors) {
        this.metrics.riskFactorCounts[factor.type]++;
      }

      if (result.payload.memoryUsage) {
        this.metrics.avgMemoryUsage = this.updateAverage(
          this.metrics.avgMemoryUsage,
          result.payload.memoryUsage.usagePercentage
        );
      }

      if (result.payload.fps !== null) {
        this.metrics.avgFps = this.updateAverage(
          this.metrics.avgFps,
          result.payload.fps
        );
      }

      this.metrics.avgDomNodeCount = this.updateAverage(
        this.metrics.avgDomNodeCount,
        result.payload.domNodeCount
      );
    } else {
      this.metrics.failureCount++;
    }
  }

  private getHealthLevel(score: number): HealthLevel {
    if (score >= 80) return 'healthy';
    if (score >= 60) return 'degraded';
    if (score >= 40) return 'unhealthy';
    return 'critical';
  }

  private updateAverage(current: number, newValue: number): number {
    const n = this.metrics.successCount;
    return current * (n - 1) / n + newValue / n;
  }

  getMetrics(): Layer3Metrics {
    return { ...this.metrics };
  }
}
```

## 2.3.12 告警规则

```typescript
interface AlertRule {
  condition: (result: Layer3Result) => boolean;
  severity: 'info' | 'warning' | 'error' | 'critical';
  message: string;
  action?: string;
}

const LAYER3_ALERT_RULES: AlertRule[] = [
  {
    condition: (r) => r.healthScore < 40,
    severity: 'critical',
    message: '系统健康度严重下降',
    action: 'restart_session'
  },
  {
    condition: (r) => r.riskFactors.some(f => f.type === 'memory_leak'),
    severity: 'error',
    message: '检测到可能的内存泄漏',
    action: 'restart_session'
  },
  {
    condition: (r) => r.riskFactors.some(f => f.type === 'error_spike' && f.severity === 'critical'),
    severity: 'error',
    message: '错误数量激增',
    action: 'investigate'
  },
  {
    condition: (r) => r.payload.fps !== null && r.payload.fps < 20,
    severity: 'warning',
    message: '界面渲染性能下降',
    action: 'reduce_load'
  }
];

function evaluateAlertRules(result: Layer3Result): AlertRule[] {
  return LAYER3_ALERT_RULES.filter(rule => rule.condition(result));
}
```
