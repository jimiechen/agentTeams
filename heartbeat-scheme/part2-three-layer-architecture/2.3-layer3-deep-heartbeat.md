# 2.3 Layer 3 深度心跳（60秒）

## 职责定位

**职责**：采集健康度指标，供可观测性使用。本身不触发恢复动作。

**目标**：全面监控、数据持久化、支持后续分析

## 采集项清单

| 指标名称 | 采集方式 | 数据类型 | 用途 |
|---------|---------|---------|------|
| `allButtons` | 遍历所有按钮 | Array | 发现未知异常按钮 |
| `sessionDuration` | 计算启动至今时间 | number | 运行时长监控 |
| `totalChars` | 累计响应字符数 | number | token 消耗估算 |
| `contextUsage` | 字符数/模型窗口 | number | 上下文使用率 |
| `lastRecoveryTime` | 上次恢复动作时间 | number | 恢复频率分析 |
| `heartbeatCount` | 心跳累计次数 | number | 健康度基线 |

## 详细采集逻辑

### 按钮快照

```javascript
// 采集所有可见按钮的 aria-label
const buttons = document.querySelectorAll('button');
const buttonSnapshot = Array.from(buttons)
    .map(btn => ({
        ariaLabel: btn.getAttribute('aria-label') || '',
        text: btn.textContent?.trim() || '',
        className: btn.className,
        isVisible: window.getComputedStyle(btn).display !== 'none'
    }))
    .filter(btn => btn.isVisible && btn.ariaLabel);
```

### 上下文使用率计算

```javascript
// 估算 context 使用率
const modelWindowSize = 8192; // 假设模型窗口大小
const contextUsage = (totalChars / modelWindowSize) * 100;
```

## CDP 执行脚本

```javascript
// Layer 3 深度探测脚本
async function layer3DeepProbe(cdp, sessionStartTime) {
    return await cdp.evaluate(`(() => {
        const buttons = document.querySelectorAll('button');
        const buttonSnapshot = Array.from(buttons)
            .filter(btn => window.getComputedStyle(btn).display !== 'none')
            .map(btn => ({
                ariaLabel: btn.getAttribute('aria-label') || '',
                text: btn.textContent?.trim() || '',
                className: btn.className.slice(0, 50)
            }))
            .filter(btn => btn.ariaLabel);
        
        // 累计响应字符数
        let totalChars = 0;
        const turns = document.querySelectorAll('.chat-turn');
        turns.forEach(turn => {
            if (!turn.classList.contains('user')) {
                totalChars += (turn.innerText || '').length;
            }
        });
        
        return {
            buttonSnapshot,
            sessionDuration: Date.now() - ${sessionStartTime},
            totalChars,
            heartbeatCount: ${heartbeatCount}
        };
    })()`);
}
```

## 成本估算

| 指标 | 数值 |
|------|------|
| 单次执行时间 | 200~300ms |
| CDP 调用次数 | 1 次 |
| 数据量 | ~2KB/条 |
| 磁盘写入频率 | 每 60 秒一次 |

## 特殊处理

### 敏感信息过滤
按钮文本中可能包含敏感信息，只采集 aria-label，不采集完整文本内容。

### 数据压缩
buttonSnapshot 只保留可见按钮，过滤隐藏按钮减少数据量。

### 采样间隔
即使 Layer 1/2 触发了恢复动作，Layer 3 仍保持 60 秒间隔，避免频繁写入。
