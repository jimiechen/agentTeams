# 8.2 环境变量管理

## 8.2.1 环境变量概述

环境变量用于存储敏感信息、部署环境差异配置和运行时覆盖。与 heartbeat.yaml 配置文件形成互补关系，实现配置的分层管理。

## 8.2.2 配置文件 vs 环境变量分工

### 8.2.2.1 分工原则

| 配置类型 | 存储位置 | 版本控制 | 示例 |
|---------|---------|---------|------|
| **业务逻辑配置** | heartbeat.yaml | ✅ 提交 | 采集间隔、恢复策略 |
| **敏感信息** | 环境变量 | ❌ 不提交 | API Key、密码 |
| **环境差异配置** | 环境变量 | ❌ 不提交 | 服务地址、调试开关 |
| **运行时覆盖** | 环境变量 | ❌ 不提交 | 临时调参、紧急开关 |

### 8.2.2.2 配置决策树

```
                    需要存储配置？
                         │
          ┌──────────────┴──────────────┐
          ▼                              ▼
    是否为敏感信息？                是否因环境而异？
          │                              │
    ┌─────┴─────┐                  ┌─────┴─────┐
    ▼           ▼                  ▼           ▼
  环境变量    是否需要              环境变量    heartbeat.yaml
             版本控制？
                │
          ┌─────┴─────┐
          ▼           ▼
    heartbeat.yaml  环境变量
```

## 8.2.3 环境变量命名规范

### 8.2.3.1 命名前缀

所有心跳系统环境变量使用 `HEARTBEAT_` 前缀：

```
HEARTBEAT_<CATEGORY>_<NAME>

示例：
HEARTBEAT_DAEMON_LAYER1_INTERVAL
HEARTBEAT_NOTIFICATION_WEBHOOK_URL
HEARTBEAT_RECOVERY_ENABLED
```

### 8.2.3.2 环境变量分类

| 分类 | 前缀 | 说明 |
|------|------|------|
| Daemon 配置 | `HEARTBEAT_DAEMON_*` | Daemon 运行参数 |
| Collector 配置 | `HEARTBEAT_COLLECTOR_*` | 采集器配置 |
| Recovery 配置 | `HEARTBEAT_RECOVERY_*` | 恢复执行器配置 |
| Notification 配置 | `HEARTBEAT_NOTIFICATION_*` | 通知渠道配置 |
| Feature 开关 | `HEARTBEAT_FEATURE_*` | 功能开关 |
| Debug 配置 | `HEARTBEAT_DEBUG_*` | 调试相关配置 |

## 8.2.5 覆盖优先级

### 8.2.5.1 优先级层级

```
优先级从高到低：

1. 命令行参数 (--config key=value)
2. 环境变量 (HEARTBEAT_*)
3. .env.{NODE_ENV} 文件
4. .env.local 文件
5. .env 文件
6. heartbeat.yaml policies.overrides
7. heartbeat.yaml policies.scenarios
8. heartbeat.yaml policies.workspaces
9. heartbeat.yaml global 默认值
```

### 8.2.5.2 优先级解析器实现

```typescript
import dotenv from 'dotenv';
import dotenvExpand from 'dotenv-expand';
import path from 'path';

interface EnvConfig {
  [key: string]: string | number | boolean;
}

class EnvPriorityResolver {
  private config: EnvConfig = {};
  
  constructor() {
    this.loadEnvFiles();
    this.loadProcessEnv();
  }
  
  private loadEnvFiles(): void {
    const nodeEnv = process.env.NODE_ENV || 'development';
    const envFiles = [
      `.env.${nodeEnv}.local`,
      `.env.${nodeEnv}`,
      '.env.local',
      '.env'
    ];
    
    for (const file of envFiles) {
      const filePath = path.resolve(process.cwd(), file);
      try {
        const result = dotenv.config({ path: filePath });
        if (!result.error) {
          dotenvExpand.expand(result);
          console.log(`[Env] Loaded ${file}`);
        }
      } catch (error) {
        console.log(`[Env] ${file} not found, skipping`);
      }
    }
  }
  
  private loadProcessEnv(): void {
    const heartbeatPrefix = 'HEARTBEAT_';
    
    for (const [key, value] of Object.entries(process.env)) {
      if (key.startsWith(heartbeatPrefix)) {
        const configKey = this.convertToConfigKey(key);
        this.config[configKey] = this.parseValue(value!);
      }
    }
  }
  
  private convertToConfigKey(envKey: string): string {
    return envKey
      .replace('HEARTBEAT_', '')
      .toLowerCase()
      .replace(/_([a-z])/g, (_, letter) => letter.toUpperCase());
  }
  
  private parseValue(value: string): string | number | boolean {
    if (value.toLowerCase() === 'true') return true;
    if (value.toLowerCase() === 'false') return false;
    if (/^\d+$/.test(value)) return parseInt(value, 10);
    if (/^\d+\.\d+$/.test(value)) return parseFloat(value);
    return value;
  }
  
  get(key: string): string | number | boolean | undefined {
    return this.config[key];
  }
  
  getAll(): EnvConfig {
    return { ...this.config };
  }
  
  applyOverrides(config: any): any {
    const result = { ...config };
    
    const envToConfigMap: Record<string, string[]> = {
      'daemonLayer1Interval': ['daemon', 'layer1Interval'],
      'daemonLayer2Interval': ['daemon', 'layer2Interval'],
      'daemonLayer3Interval': ['daemon', 'layer3Interval'],
      'daemonMaxRecoveryRetries': ['daemon', 'maxRecoveryRetries'],
      'daemonMetricsEnabled': ['daemon', 'metricsEnabled'],
      'daemonPersistenceEnabled': ['daemon', 'persistenceEnabled'],
      'daemonPersistencePath': ['daemon', 'persistencePath'],
      'recoveryEnabled': ['recovery', 'enabled'],
      'recoveryDebounceMs': ['recovery', 'debounceMs'],
      'recoveryMaxConcurrent': ['recovery', 'maxConcurrent'],
      'recoveryAuditEnabled': ['recovery', 'auditEnabled']
    };
    
    for (const [envKey, configPath] of Object.entries(envToConfigMap)) {
      const envValue = this.config[envKey];
      if (envValue !== undefined) {
        this.setNestedValue(result, configPath, envValue);
      }
    }
    
    return result;
  }
  
  private setNestedValue(obj: any, path: string[], value: any): void {
    let current = obj;
    for (let i = 0; i < path.length - 1; i++) {
      if (!current[path[i]]) {
        current[path[i]] = {};
      }
      current = current[path[i]];
    }
    current[path[path.length - 1]] = value;
  }
}
```

### 8.2.6.2 验证器实现

```typescript
interface ValidationResult {
  valid: boolean;
  errors: string[];
  warnings: string[];
}

class EnvValidator {
  validate(rules: EnvValidationRule[] = ENV_VALIDATION_RULES): ValidationResult {
    const errors: string[] = [];
    const warnings: string[] = [];
    
    for (const rule of rules) {
      const value = process.env[rule.name];
      
      if (value === undefined || value === '') {
        if (rule.required) {
          errors.push(`${rule.name} is required but not set`);
        }
        continue;
      }
      
      if (rule.sensitive) {
        this.validateSensitiveValue(rule, value, warnings);
      }
      
      switch (rule.type) {
        case 'number':
          this.validateNumber(rule, value, errors);
          break;
        case 'boolean':
          this.validateBoolean(rule, value, errors);
          break;
        case 'url':
          this.validateUrl(rule, value, errors);
          break;
        case 'string':
          this.validateString(rule, value, errors);
          break;
      }
    }
    
    return {
      valid: errors.length === 0,
      errors,
      warnings
    };
  }
  
  private validateNumber(rule: EnvValidationRule, value: string, errors: string[]): void {
    const num = parseInt(value, 10);
    
    if (isNaN(num)) {
      errors.push(`${rule.name} must be a number, got: ${value}`);
      return;
    }
    
    if (rule.min !== undefined && num < rule.min) {
      errors.push(`${rule.name} must be >= ${rule.min}, got: ${num}`);
    }
    
    if (rule.max !== undefined && num > rule.max) {
      errors.push(`${rule.name} must be <= ${rule.max}, got: ${num}`);
    }
  }
  
  private validateBoolean(rule: EnvValidationRule, value: string, errors: string[]): void {
    if (!['true', 'false', '1', '0'].includes(value.toLowerCase())) {
      errors.push(`${rule.name} must be a boolean (true/false), got: ${value}`);
    }
  }
  
  private validateUrl(rule: EnvValidationRule, value: string, errors: string[]): void {
    try {
      new URL(value);
    } catch {
      errors.push(`${rule.name} must be a valid URL, got: ${value}`);
    }
  }
  
  private validateString(rule: EnvValidationRule, value: string, errors: string[]): void {
    if (rule.pattern && !rule.pattern.test(value)) {
      errors.push(`${rule.name} does not match pattern ${rule.pattern}`);
    }
  }
  
  private validateSensitiveValue(rule: EnvValidationRule, value: string, warnings: string[]): void {
    if (value.length < 8) {
      warnings.push(`${rule.name} appears to be too short for a sensitive value`);
    }
    
    if (value.includes('test') || value.includes('dummy')) {
      warnings.push(`${rule.name} appears to contain test/dummy value`);
    }
  }
}
```

## 8.2.7 敏感信息处理

### 8.2.7.1 敏感信息检测

```typescript
class SensitiveValueDetector {
  private sensitivePatterns = [
    /password/i,
    /secret/i,
    /token/i,
    /key/i,
    /credential/i,
    /api[_-]?key/i
  ];
  
  detect(envKey: string): boolean {
    return this.sensitivePatterns.some(pattern => pattern.test(envKey));
  }
  
  mask(value: string): string {
    if (value.length <= 4) {
      return '****';
    }
    return value.slice(0, 2) + '****' + value.slice(-2);
  }
}

class SecureEnvLogger {
  private detector = new SensitiveValueDetector();
  
  logEnvSummary(): void {
    console.log('[Env] Environment variables summary:');
    
    for (const [key, value] of Object.entries(process.env)) {
      if (key.startsWith('HEARTBEAT_')) {
        if (this.detector.detect(key)) {
          console.log(`  ${key}=${this.detector.mask(value!)}`);
        } else {
          console.log(`  ${key}=${value}`);
        }
      }
    }
  }
}
```

## 8.2.8 环境变量注入到配置

### 8.2.8.1 配置注入器

```typescript
class ConfigInjector {
  private envResolver: EnvPriorityResolver;
  
  constructor() {
    this.envResolver = new EnvPriorityResolver();
  }
  
  inject(yamlConfig: HeartbeatConfig): HeartbeatConfig {
    const envOverrides = this.envResolver.getAll();
    
    let config = JSON.parse(JSON.stringify(yamlConfig));
    
    config = this.envResolver.applyOverrides(config);
    
    config = this.resolveEnvReferences(config);
    
    return config;
  }
  
  private resolveEnvReferences(config: any): any {
    const resolved = JSON.stringify(config);
    
    const result = resolved.replace(
      /\$\{([^}]+)\}/g,
      (_, envVar) => {
        const value = process.env[envVar];
        if (value === undefined) {
          console.warn(`[Config] Environment variable ${envVar} is not set`);
          return '';
        }
        return value;
      }
    );
    
    return JSON.parse(result);
  }
}
```
