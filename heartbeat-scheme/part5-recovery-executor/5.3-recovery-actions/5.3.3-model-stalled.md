# 5.3.3 模型停滞：stop + retry（重试次数限制）

## 实现代码

```typescript
private async recoverModelStalled(signal: HeartbeatSignal): Promise<RecoveryResult> {
  const retryCount = this.retryCountByMode.get('model-stalled') ?? 0;

  if (retryCount >= this.policy.stalled_retry_limit) {
    await this.notifyAlert('model-stalled',
      `停滞 ${signal.stalledSeconds}s，已重试 ${retryCount} 次失败，请人工介入`);
    return { attempted: false, action: 'retry-exhausted', success: false };
  }

  // Step 1: 点击停止
  const stopped = await this.clickStopButton();
  if (!stopped) return { attempted: true, action: 'stop-failed', success: false };

  // Step 2: 等待 3 秒让状态稳定
  await sleep(3000);

  // Step 3: 点击重试
  const retried = await this.clickRetryButton();
  this.retryCountByMode.set('model-stalled', retryCount + 1);

  if (retried) {
    await this.notifyRecovery('model-stalled', `stop + retry（第 ${retryCount + 1} 次）`);
    return { attempted: true, action: 'stop+retry', success: true };
  }
  return { attempted: true, action: 'retry-failed', success: false };
}

private async clickStopButton(): Promise<boolean> {
  return this.cdp.evaluate<boolean>(`(() => {
    const buttons = document.querySelectorAll('button');
    for (const btn of buttons) {
      const label = btn.getAttribute('aria-label') || '';
      const text = btn.textContent?.trim() || '';
      if (label.includes('停止') || text.includes('停止') || label.includes('Stop')) {
        btn.click();
        return true;
      }
    }
    return false;
  })()`);
}

private async clickRetryButton(): Promise<boolean> {
  return this.cdp.evaluate<boolean>(`(() => {
    const buttons = document.querySelectorAll('button');
    for (const btn of buttons) {
      const label = btn.getAttribute('aria-label') || '';
      const text = btn.textContent?.trim() || '';
      if (label.includes('重试') || text.includes('重试') || label.includes('Retry')) {
        btn.click();
        return true;
      }
    }
    return false;
  })()`);
}
```

## 执行流程

```
收到 model-stalled 信号
    │
    ▼
检查重试次数是否超过限制
    │
    ├─→ 超过限制 → 发送告警，不执行恢复
    │
    └─→ 未超过 → Step 1: 点击停止按钮
            │
            ▼
        Step 2: 等待 3 秒
            │
            ▼
        Step 3: 点击重试按钮
            │
            ▼
        增加重试计数
            │
            ▼
        发送恢复通知
```

## 为什么等待 3 秒

Trae 的停止操作需要时间让模型真正释放资源，立即 retry 可能触发新的错误。3 秒是经验值，可以通过配置调整。

## 重试计数策略

- 每种模式独立计数
- 计数器在恢复成功后重置
- 超过重试上限后转为 task-interrupted

## 恢复验证

恢复执行后，下一次心跳检查：
- `stalledSeconds` 是否重置为 0
- `charsDelta` 是否大于 0（文本开始增长）
