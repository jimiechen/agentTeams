# 5.3.2 Modal Blocking 恢复动作

## 5.3.2.1 问题定义

**modal-blocking** 是指 UI 层出现模态弹窗（如"保留/删除"确认框）导致任务无法继续执行的状态。这是一种显式的 UI 阻塞，需要用户交互才能解除。

### 识别特征

| 特征 | 说明 |
|------|------|
| **触发条件** | 检测到"保留/删除"按钮对 |
| **检测层级** | Layer 1（快速心跳） |
| **时效要求** | 5 秒内识别 |
| **自治可行性** | ✅ 按策略点击按钮 |

### 常见模态弹窗类型

```typescript
enum ModalType {
  RETAIN_DELETE = 'retain-delete',

interface ModalSignal {
```

## 5.3.2.4 恢复执行器实现

```typescript
class ModalBlockingRecoveryExecutor implements ActionExecutor {
  readonly type = 'modal-blocking';
  readonly action = 'click-modal-button';
  
  private page: Page;
  private buttonFinder: ButtonFinder;
  private permissionChecker: PermissionChecker;
  private strategyManager: StrategyManager;
  
  constructor(page: Page) {
    this.page = page;
    this.buttonFinder = new ButtonFinder(page);
    this.permissionChecker = new PermissionChecker();
    this.strategyManager = new DefaultStrategyManager();
  }

  async execute(context: RecoveryContext): Promise<RecoveryResult> {
    const startTime = Date.now();
    
    console.log(`[ModalBlocking] 开始恢复工作区 ${context.workspaceId}`);
    
    const modalInfo = await this.detectModal();
    
    if (!modalInfo.detected) {
      return {
        success: false,
        action: this.action,
        reason: '未检测到模态弹窗',
        durationMs: Date.now() - startTime,
        nextState: 'healthy'
      };
    }
    
    const strategy = this.strategyManager.getStrategy(context.workspaceId);
    const actionToTake = this.determineAction(strategy, modalInfo);
    
    const permissionResult = this.permissionChecker.checkPermission(actionToTake);
    if (!permissionResult.allowed) {
      return {
        success: false,
        action: actionToTake,
        reason: permissionResult.reason,
        durationMs: Date.now() - startTime,
        nextState: 'modal-blocking'
      };
    }
    
    try {
      await this.clickButton(actionToTake);
      
      this.permissionChecker.recordClick(actionToTake);
      
      console.log(`[ModalBlocking] 已点击"${actionToTake}"按钮`);
      
      return {
        success: true,
        action: `click-${actionToTake}`,
        reason: `成功点击"${actionToTake}"按钮`,
        durationMs: Date.now() - startTime,
        nextState: 'verifying'
      };
    } catch (error) {
      return {
        success: false,
        action: actionToTake,
        reason: `点击失败: ${error instanceof Error ? error.message : '未知错误'}`,
        durationMs: Date.now() - startTime,
        nextState: 'modal-blocking'
      };
    }
  }

  private async detectModal(): Promise<ModalDetectionResult> {
    for (const selector of MODAL_BLOCKING_SELECTORS.modalContainer) {
      try {
        const modal = this.page.locator(selector).first();
        const isVisible = await modal.isVisible({ timeout: 1000 });
        
        if (isVisible) {
          const hasRetain = await this.buttonFinder.findButton('retain');
          const hasDelete = await this.buttonFinder.findButton('delete');
          
          return {
            detected: true,
            modalType: ModalType.RETAIN_DELETE,
            hasRetainBtn: hasRetain.found,
            hasDeleteBtn: hasDelete.found
          };
        }
      } catch {
        continue;
      }
    }
    
    return { detected: false };
  }

  private determineAction(
    strategy: WorkspaceStrategy,
    modalInfo: ModalDetectionResult
  ): string {
    const config = strategy.config as PMCLIStrategyConfig | DEVCLIStrategyConfig;
    
    if ('modalDefaultAction' in config) {
      return config.modalDefaultAction;
    }
    
    return 'retain';
  }

  private async clickButton(action: string): Promise<void> {
    const result = await this.buttonFinder.findButton(action);
    
    if (!result.found || !result.element) {
      throw new Error(`未找到"${action}"按钮`);
    }
    
    await result.element.click();
  }

  async canExecute(context: RecoveryContext): Promise<boolean> {
    const modalInfo = await this.detectModal();
    return modalInfo.detected;
  }
}

interface ModalDetectionResult {
  detected: boolean;
  modalType?: ModalType;
  hasRetainBtn?: boolean;
  hasDeleteBtn?: boolean;
}
```

## 5.3.2.5 PMCLI/DEVCLI 策略差异

### PMCLI 策略

```typescript
const PMCLI_MODAL_CONFIG = {
  defaultAction: 'retain',
  autoExecute: true,
  requiresConfirmation: false,
  maxRetries: 5,
  verifyingWindowMs: 30000,
  alertOnAction: false,
  logLevel: 'info'
};
```

**PMCLI 特点**：
- 默认点击"保留"按钮
- 自动执行，无需确认
- 不发送告警
- 适合产品需求文档等需要保留历史的场景

### DEVCLI 策略

```typescript
const DEVCLI_MODAL_CONFIG = {
  defaultAction: 'delete',
  autoExecute: true,
  requiresConfirmation: false,
  maxRetries: 5,
  verifyingWindowMs: 30000,
  alertOnAction: true,
  logLevel: 'warning',
  preserveCodeContext: true
};
```

**DEVCLI 特点**：
- 默认点击"删除"按钮（清理临时文件）
- 自动执行
- 发送告警通知
- 保留代码上下文

## 5.3.2.6 验证逻辑

```typescript
class ModalBlockingVerifier {
  private readonly VERIFY_WINDOW_MS = 30000;
  private readonly CHECK_INTERVAL_MS = 2000;
  
  async verify(page: Page): Promise<boolean> {
    const startTime = Date.now();
    
    while (Date.now() - startTime < this.VERIFY_WINDOW_MS) {
      const modalGone = await this.isModalGone(page);
      
      if (modalGone) {
        console.log('[ModalBlocking] 弹窗已消失，恢复成功');
        return true;
      }
      
      await this.sleep(this.CHECK_INTERVAL_MS);
    }
    
    console.log('[ModalBlocking] 验证窗口超时，弹窗仍存在');
    return false;
  }
  
  private async isModalGone(page: Page): Promise<boolean> {
    for (const selector of MODAL_BLOCKING_SELECTORS.modalContainer) {
      try {
        const modal = page.locator(selector).first();
        const isVisible = await modal.isVisible({ timeout: 1000 });
        
        if (isVisible) {
          return false;
        }
      } catch {
        continue;
      }
    }
    
    return true;
  }
  
  private sleep(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

## 5.3.2.7 特殊弹窗处理

### 文件覆盖确认

```typescript
class OverwriteModalHandler {
  async handle(page: Page, context: RecoveryContext): Promise<RecoveryResult> {
    console.log('[ModalBlocking] 处理文件覆盖确认弹窗');
    
    const strategy = this.strategyManager.getStrategy(context.workspaceId);
    
    if (strategy.type === WorkspaceType.DEVCLI) {
      return await this.clickConfirm(page);
    } else {
      return await this.clickCancel(page);
    }
  }
  
  private async clickConfirm(page: Page): Promise<RecoveryResult> {
    const result = await this.buttonFinder.findButton('confirm');
    
    if (result.found && result.element) {
      await result.element.click();
      return {
        success: true,
        action: 'click-confirm',
        reason: '确认覆盖文件',
        durationMs: 0,
        nextState: 'verifying'
      };
    }
    
    return {
      success: false,
      action: 'none',
      reason: '未找到确认按钮',
      durationMs: 0,
      nextState: 'modal-blocking'
    };
  }
  
  private async clickCancel(page: Page): Promise<RecoveryResult> {
    const result = await this.buttonFinder.findButton('cancel');
    
    if (result.found && result.element) {
      await result.element.click();
      return {
        success: true,
        action: 'click-cancel',
        reason: '取消覆盖操作',
        durationMs: 0,
        nextState: 'verifying'
      };
    }
    
    return {
      success: false,
      action: 'none',
      reason: '未找到取消按钮',
      durationMs: 0,
      nextState: 'modal-blocking'
    };
  }
}
```

## 5.3.2.8 日志与监控

```typescript
interface ModalBlockingLog {
  timestamp: string;
  workspaceId: string;
  taskId: string;
  modalType: ModalType;
  action: string;
  result: 'success' | 'failed' | 'skipped';
  reason: string;
  durationMs: number;
  modalSnapshot?: {
    title: string;
    message: string;
  };
}

class ModalBlockingLogger {
  private logs: ModalBlockingLog[] = [];
  
  log(entry: Omit<ModalBlockingLog, 'timestamp'>): void {
    const log: ModalBlockingLog = {
      ...entry,
      timestamp: new Date().toISOString()
    };
    
    this.logs.push(log);
    this.writeToFile(log);
    
    if (entry.result === 'failed') {
      this.sendAlert(log);
    }
  }
  
  private writeToFile(log: ModalBlockingLog): void {
    const logPath = `logs/modal-blocking-${new Date().toISOString().split('T')[0]}.jsonl`;
    fs.appendFileSync(logPath, JSON.stringify(log) + '\n');
  }
  
  private sendAlert(log: ModalBlockingLog): void {
    console.error(`[ModalBlocking 告警] 工作区 ${log.workspaceId} 恢复失败: ${log.reason}`);
  }
  
  getStats(): {
    total: number;
    byType: Record<ModalType, number>;
    byAction: Record<string, number>;
    successRate: number;
  } {
    const total = this.logs.length;
    const success = this.logs.filter(l => l.result === 'success').length;
    
    const byType: Record<ModalType, number> = {} as any;
    const byAction: Record<string, number> = {};
    
    for (const log of this.logs) {
      byType[log.modalType] = (byType[log.modalType] || 0) + 1;
      byAction[log.action] = (byAction[log.action] || 0) + 1;
    }
    
    return {
      total,
      byType,
      byAction,
      successRate: total > 0 ? success / total : 0
    };
  }
}
```
