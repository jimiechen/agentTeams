# 5.2 工作区策略配置

## 5.2.1 策略配置概述

不同的工作区类型（PMCLI、DEVCLI）有不同的恢复策略需求。工作区策略配置系统允许为不同类型的工作区定义差异化的恢复行为。

核心概念：
- **工作区类型**：PMCLI（产品管理）、DEVCLI（开发）
- **策略模板**：预定义的恢复策略配置
- **策略继承**：支持从全局配置继承并覆盖
- **动态切换**：运行时可以切换策略

### 5.2.2.2 工作区类型检测

```typescript
class WorkspaceTypeDetector {
  private readonly PMCLI_PATTERNS = [
    /product/i,
    /requirement/i,
    /specification/i,
    /prd/i,
    /roadmap/i,
    /feature/i
  ];
  
  private readonly DEVCLI_PATTERNS = [
    /develop/i,
    /implement/i,
    /code/i,
    /fix/i,
    /refactor/i,
    /test/i,
    /debug/i
  ];

  detectType(workspaceName: string, recentTasks: string[]): WorkspaceType {
    const allText = [workspaceName, ...recentTasks].join(' ').toLowerCase();
    
    const pmcliScore = this.calculateScore(allText, this.PMCLI_PATTERNS);
    const devcliScore = this.calculateScore(allText, this.DEVCLI_PATTERNS);
    
    if (pmcliScore > devcliScore && pmcliScore > 0) {
      return WorkspaceType.PMCLI;
    }
    
    if (devcliScore > 0) {
      return WorkspaceType.DEVCLI;
    }
    
    return WorkspaceType.UNKNOWN;
  }

  private calculateScore(text: string, patterns: RegExp[]): number {
    let score = 0;
    
    for (const pattern of patterns) {
      const matches = text.match(pattern);
      if (matches) {
        score += matches.length;
      }
    }
    
    return score;
  }
}
```

## 5.2.3 PMCLI/DEVCLI 策略差异

### 5.2.3.1 策略差异对比表

| 配置项 | PMCLI 策略 | DEVCLI 策略 | 差异原因 |
|--------|-----------|------------|---------|
| **stalled 阈值** | 45 秒 | 30 秒 | PMCLI 任务通常更长 |
| **重试次数** | 2 次 | 3 次 | DEVCLI 任务更关键 |
| **后台运行** | 自动点击 | 需确认 | PMCLI 更注重流程 |
| **删除弹窗** | 默认保留 | 默认删除 | DEVCLI 需要清理 |
| **告警级别** | warning | critical | DEVCLI 需更快响应 |
| **验证窗口** | 90 秒 | 60 秒 | PMCLI 响应较慢 |
| **冷却时间** | 15 秒 | 10 秒 | DEVCLI 需更快恢复 |

### 5.2.4.2 策略管理器实现

```typescript
class DefaultStrategyManager implements StrategyManager {
  private strategies: Map<string, WorkspaceStrategy> = new Map();
  private typeDetector: WorkspaceTypeDetector;
  
  constructor() {
    this.typeDetector = new WorkspaceTypeDetector();
  }
  
  getStrategy(workspaceId: string): WorkspaceStrategy {
    const cached = this.strategies.get(workspaceId);
    
    if (cached) {
      return cached;
    }
    
    const type = this.detectWorkspaceType(workspaceId);
    const defaultStrategy = this.getDefaultStrategy(type);
    
    const strategy: WorkspaceStrategy = {
      workspaceId,
      type,
      config: defaultStrategy.config,
      overrides: {},
      createdAt: new Date().toISOString(),
      updatedAt: new Date().toISOString()
    };
    
    this.strategies.set(workspaceId, strategy);
    return strategy;
  }
  
  setStrategy(workspaceId: string, strategy: WorkspaceStrategy): void {
    strategy.updatedAt = new Date().toISOString();
    this.strategies.set(workspaceId, strategy);
    this.persistStrategy(strategy);
  }
  
  getDefaultStrategy(type: WorkspaceType): WorkspaceStrategy {
    const configs: Record<WorkspaceType, PMCLIStrategyConfig | DEVCLIStrategyConfig> = {
      [WorkspaceType.PMCLI]: PMCLI_STRATEGY,
      [WorkspaceType.DEVCLI]: DEVCLI_STRATEGY,
      [WorkspaceType.UNKNOWN]: PMCLI_STRATEGY
    };
    
    return {
      workspaceId: 'default',
      type,
      config: configs[type],
      overrides: {},
      createdAt: new Date().toISOString(),
      updatedAt: new Date().toISOString()
    };
  }
  
  resetToDefault(workspaceId: string): void {
    const type = this.detectWorkspaceType(workspaceId);
    const defaultStrategy = this.getDefaultStrategy(type);
    
    defaultStrategy.workspaceId = workspaceId;
    this.strategies.set(workspaceId, defaultStrategy);
  }
  
  private detectWorkspaceType(workspaceId: string): WorkspaceType {
    return WorkspaceType.UNKNOWN;
  }
  
  private persistStrategy(strategy: WorkspaceStrategy): void {
    const path = `config/strategies/${strategy.workspaceId}.json`;
    fs.writeFileSync(path, JSON.stringify(strategy, null, 2));
  }
}
```

## 5.2.5 策略覆盖机制

### 5.2.5.1 覆盖配置

```typescript
interface StrategyOverride {
  workspaceId: string;
  overrides: {
    stalledThreshold?: number;
    maxRetries?: number;
    autoBackground?: boolean;
    modalDefaultAction?: 'retain' | 'delete';
    alertLevel?: 'info' | 'warning' | 'critical';
    verifyingWindowMs?: number;
    cooldownMs?: number;
    enableAutoRecovery?: boolean;
  };
  reason: string;
  expiresAt?: string;
}

class StrategyOverrideManager {
  private overrides: Map<string, StrategyOverride> = new Map();
  
  applyOverride(workspaceId: string, override: StrategyOverride): void {
    this.overrides.set(workspaceId, override);
    
    if (override.expiresAt) {
      const delay = new Date(override.expiresAt).getTime() - Date.now();
      setTimeout(() => {
        this.removeOverride(workspaceId);
      }, delay);
    }
  }
  
  removeOverride(workspaceId: string): void {
    this.overrides.delete(workspaceId);
  }
  
  getOverride(workspaceId: string): StrategyOverride | null {
    const override = this.overrides.get(workspaceId);
    
    if (override?.expiresAt && new Date(override.expiresAt) < new Date()) {
      this.overrides.delete(workspaceId);
      return null;
    }
    
    return override || null;
  }
  
  mergeWithOverride(
    baseStrategy: WorkspaceStrategy,
    override: StrategyOverride
  ): WorkspaceStrategy {
    return {
      ...baseStrategy,
      config: {
        ...baseStrategy.config,
        ...override.overrides
      },
      overrides: override.overrides
    };
  }
}
```

```typescript
    };
    
    this.overrideManager.applyOverride(workspaceId, override);
  }
  
  restoreNormal(workspaceId: string): void {
    this.overrideManager.removeOverride(workspaceId);
  }
}
```

## 5.2.6 策略执行器

### 5.2.6.1 策略执行器接口

```typescript
  action: string;

  durationMs: number;
  nextState: HealthMode;
}
```

### 5.2.6.2 策略执行器实现

```typescript
class DefaultStrategyExecutor implements StrategyExecutor {
  private strategyManager: StrategyManager;
  private overrideManager: StrategyOverrideManager;
  private actionExecutors: Map<string, ActionExecutor>;
  
  constructor() {
    this.strategyManager = new DefaultStrategyManager();
    this.overrideManager = new StrategyOverrideManager();
    this.actionExecutors = new Map();
    this.registerActionExecutors();
  }
  
  private registerActionExecutors(): void {
    this.actionExecutors.set('click-background', new ClickBackgroundExecutor());
    this.actionExecutors.set('click-retain', new ClickRetainExecutor());
    this.actionExecutors.set('click-delete', new ClickDeleteExecutor());
    this.actionExecutors.set('stop-retry', new StopRetryExecutor());
  }
  
  async execute(
    workspaceId: string,
    mode: HealthMode,
    context: RecoveryContext
  ): Promise<RecoveryResult> {
    const startTime = Date.now();
    
    let strategy = this.strategyManager.getStrategy(workspaceId);
    
    const override = this.overrideManager.getOverride(workspaceId);
    if (override) {
      strategy = this.overrideManager.mergeWithOverride(strategy, override);
    }
    
    const actionConfig = this.findActionConfig(strategy, mode);
    
    if (!actionConfig) {
      return {
        success: false,
        action: 'none',
        reason: `未找到 ${mode} 的恢复动作配置`,
        durationMs: Date.now() - startTime,
        nextState: mode
      };
    }
    
    if (!actionConfig.autoExecute && !actionConfig.requiresConfirmation) {
      return {
        success: false,
        action: 'none',
        reason: '恢复动作需要手动执行',
        durationMs: Date.now() - startTime,
        nextState: mode
      };
    }
    
    const executor = this.actionExecutors.get(actionConfig.action);
    
    if (!executor) {
      return {
        success: false,
        action: actionConfig.action,
        reason: `未找到执行器: ${actionConfig.action}`,
        durationMs: Date.now() - startTime,
        nextState: mode
      };
    }
    
    try {
      const result = await executor.execute(context);
      
      return {
        ...result,
        durationMs: Date.now() - startTime
      };
    } catch (error) {
      return {
        success: false,
        action: actionConfig.action,
        reason: error instanceof Error ? error.message : '执行失败',
        durationMs: Date.now() - startTime,
        nextState: mode
      };
    }
  }
  
  private findActionConfig(
    strategy: WorkspaceStrategy,
    mode: HealthMode
  ): RecoveryActionConfig | null {
    const config = strategy.config as PMCLIStrategyConfig | DEVCLIStrategyConfig;
    
    return config.recoveryActions.find(action => action.type === mode) || null;
  }
}
```

### 5.2.7.2 REST API 实现

```typescript
class StrategyAPIHandler {
  private strategyManager: StrategyManager;
  private overrideManager: StrategyOverrideManager;
  
  async handleGetStrategy(req: Request, res: Response): Promise<void> {
    const { workspaceId } = req.params;
    
    const strategy = this.strategyManager.getStrategy(workspaceId);
    
    res.json({
      success: true,
      data: strategy
    });
  }
  
  async handleUpdateStrategy(req: Request, res: Response): Promise<void> {
    const { workspaceId } = req.params;
    const updates = req.body;
    
    const current = this.strategyManager.getStrategy(workspaceId);
    const updated: WorkspaceStrategy = {
      ...current,
      config: {
        ...current.config,
        ...updates
      },
      updatedAt: new Date().toISOString()
    };
    
    this.strategyManager.setStrategy(workspaceId, updated);
    
    res.json({
      success: true,
      data: updated
    });
  }
  
  async handleResetStrategy(req: Request, res: Response): Promise<void> {
    const { workspaceId } = req.params;
    
    this.strategyManager.resetToDefault(workspaceId);
    
    res.json({
      success: true,
      message: `工作区 ${workspaceId} 策略已重置`
    });
  }
  
  async handleApplyOverride(req: Request, res: Response): Promise<void> {
    const { workspaceId } = req.params;
    const override: StrategyOverride = req.body;
    
    this.overrideManager.applyOverride(workspaceId, override);
    
    res.json({
      success: true,
      message: `已应用临时覆盖`
    });
  }
}
```

## 5.2.9 策略监控与告警

### 5.2.9.1 策略变更监控

```typescript
class StrategyMonitor {
  private changeLog: StrategyChangeLogEntry[] = [];
  
  logChange(entry: StrategyChangeLogEntry): void {
    this.changeLog.push({
      ...entry,
      timestamp: new Date().toISOString()
    });
    
    if (this.shouldAlert(entry)) {
      this.sendAlert(entry);
    }
  }
  
  private shouldAlert(entry: StrategyChangeLogEntry): boolean {
    const criticalChanges = [
      'enableAutoRecovery',
      'maxRetries',
      'alertLevel'
    ];
    
    return criticalChanges.some(field => 
      entry.changes.some(change => change.field === field)
    );
  }
  
  private sendAlert(entry: StrategyChangeLogEntry): void {
    console.log(`[策略告警] 工作区 ${entry.workspaceId} 发生关键配置变更`);
  }
  
  getChangeLog(workspaceId?: string): StrategyChangeLogEntry[] {
    if (workspaceId) {
      return this.changeLog.filter(e => e.workspaceId === workspaceId);
    }
    return [...this.changeLog];
  }
}

interface StrategyChangeLogEntry {
  timestamp: string;
  workspaceId: string;
  changedBy: string;
  changes: Array<{
    field: string;
    oldValue: any;
    newValue: any;
  }>;
  reason: string;
}
```
