# 4.2 Layer 1 快速探测脚本

## 设计目标

单次 CDP `evaluate` 调用，一次性返回所有 Layer 1 信号，避免多次 CDP 往返。

## CDP 脚本

```javascript
(() => {
  const result = {
    taskStatus: 'unknown',
    taskStatusText: '',
    hasBackgroundButton: false,
    hasCancelButton: false,
    hasKeepButton: false,
    hasDeleteButton: false,
  };

  // 任务状态（精确匹配当前工作区名）
  const items = document.querySelectorAll('.index-module__task-item___zOpfg');
  for (const item of items) {
    const text = item.textContent || '';
    if (text.includes('${workspace}')) {
      if (text.includes('完成')) result.taskStatus = 'completed';
      else if (text.includes('进行中')) result.taskStatus = 'in_progress';
      else if (text.includes('中断')) result.taskStatus = 'interrupted';
      else if (text.includes('等待')) result.taskStatus = 'waiting';
      result.taskStatusText = text.slice(0, 50);
      break;
    }
  }

  // 按钮扫描（一次遍历同时判定四种按钮）
  const buttons = document.querySelectorAll('button');
  for (const btn of buttons) {
    const label = btn.getAttribute('aria-label') || '';
    const text = btn.textContent?.trim() || '';
    if (label.includes('后台') || text.includes('后台运行')) result.hasBackgroundButton = true;
    if (label === '取消' || text === '取消') result.hasCancelButton = true;
    if (label.includes('保留') || text.includes('保留')) result.hasKeepButton = true;
    if (label.includes('删除') || text.includes('删除')) result.hasDeleteButton = true;
  }

  return result;
})()
```

## 关键技术点

### 模板注入
`${workspace}` 在 TypeScript 侧用字符串拼接，避免运行时参数传递：

```typescript
const script = SCRIPT_LAYER1.replace('${workspace}', this.workspace);
const result = await this.cdp.evaluate(script);
```

### 性能优化
- 使用 `for...of` 循环而非 `forEach`，在找到目标后可以 `break`
- 一次遍历同时检查四种按钮状态，避免多次 DOM 查询

### 双重匹配
同时检查 `aria-label` 和 `textContent`：
- `aria-label` 更可靠（机器可读）
- `textContent` 作为备选（某些按钮可能没有 aria-label）

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `taskStatus` | string | 任务状态（unknown/completed/in_progress/interrupted/waiting） |
| `taskStatusText` | string | 原始状态文本（前 50 字符） |
| `hasBackgroundButton` | boolean | 是否存在"后台运行"按钮 |
| `hasCancelButton` | boolean | 是否存在"取消"按钮 |
| `hasKeepButton` | boolean | 是否存在"保留"按钮 |
| `hasDeleteButton` | boolean | 是否存在"删除"按钮 |

## 任务状态判定优先级

```
完成 > 进行中 > 中断 > 等待 > unknown
```

一旦匹配到更高优先级的状态，立即停止检查。
