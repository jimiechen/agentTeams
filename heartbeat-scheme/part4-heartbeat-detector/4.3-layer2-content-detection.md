# 4.3 Layer 2 内容探测脚本

## 概述

Layer 2 内容探测通过 CDP (Chrome DevTools Protocol) 在浏览器上下文中执行脚本，检查页面的内容状态、网络活动和用户交互痕迹。该层能够获取比 Layer 1 更详细的页面状态信息。

## 设计目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 执行时间 | 100ms - 1000ms | 中等耗时 |
| 准确率 | > 90% | 高准确度模式判定 |
| 干扰性 | 低 | 不影响页面正常运行 |
| 检测频率 | 每 5-10 秒 | 中等频率 |

## CDP 脚本执行框架

### contentCheck() 实现

```typescript
class HeartbeatDetector {
  private cdpClient: CDPClient;
  private contentCheckScript: string;

  async contentCheck(): Promise<DetectionResult> {
    const startTime = performance.now();
    const signals: Signal[] = [];

    try {
      const scriptResult = await this.executeCDPScript(this.contentCheckScript);
      
      if (scriptResult.domActivity) {
        signals.push({
          type: 'dom_changed',
          source: 'layer2',
          value: scriptResult.domActivity,
          timestamp: Date.now(),
          weight: 0.7
        });
      }

      if (scriptResult.networkActivity) {
        signals.push({
          type: 'network_active',
          source: 'layer2',
          value: scriptResult.networkActivity,
          timestamp: Date.now(),
          weight: 0.8
        });
      }

      if (scriptResult.userActivity) {
        signals.push({
          type: 'user_interaction',
          source: 'layer2',
          value: scriptResult.userActivity,
          timestamp: Date.now(),
          weight: 0.9
        });
      }

      if (scriptResult.memoryPressure) {
        signals.push({
          type: 'memory_pressure',
          source: 'layer2',
          value: scriptResult.memoryPressure,
          timestamp: Date.now(),
          weight: 0.6
        });
      }

      const mode = this.determineModeFromSignals(signals);
      const confidence = this.calculateConfidence(signals);

      return {
        mode,
        confidence,
        signals,
        timestamp: Date.now(),
        layer: 2
      };

    } catch (error) {
      return this.handleContentCheckError(error as Error);
    }
  }

  private async executeCDPScript(script: string): Promise<ContentCheckResult> {
    const result = await this.cdpClient.Runtime.evaluate({
      expression: script,
      returnByValue: true,
      timeout: this.config.layer2Timeout
    });

    return result.result.value as ContentCheckResult;
  }
}

interface ContentCheckResult {
  domActivity: DOMActivityInfo;
  networkActivity: NetworkActivityInfo;
  userActivity: UserActivityInfo;
  memoryPressure: MemoryPressureInfo;
  timestamp: number;
}
```

## CDP 检测脚本

### 完整内容检测脚本

```javascript
(function() {
  const result = {
    domActivity: null,
    networkActivity: null,
    userActivity: null,
    memoryPressure: null,
    timestamp: Date.now()
  };

  result.domActivity = (function() {
    const body = document.body;
    if (!body) return null;

    return {
      childCount: body.children.length,
      totalNodes: document.getElementsByTagName('*').length,
      textContent: body.innerText ? body.innerText.length : 0,
      hasIframes: document.getElementsByTagName('iframe').length > 0,
      hasScripts: document.getElementsByTagName('script').length > 0,
      forms: document.forms.length,
      inputs: document.getElementsByTagName('input').length,
      buttons: document.getElementsByTagName('button').length
    };
  })();

  result.networkActivity = (function() {
    const entries = performance.getEntriesByType('resource');
    const recentEntries = entries.filter(e => 
      Date.now() - e.startTime < 5000
    );

    return {
      totalResources: entries.length,
      recentResources: recentEntries.length,
      resourceTypes: {
        script: recentEntries.filter(e => e.initiatorType === 'script').length,
        link: recentEntries.filter(e => e.initiatorType === 'link').length,
        img: recentEntries.filter(e => e.initiatorType === 'img').length,
        fetch: recentEntries.filter(e => e.initiatorType === 'fetch').length,
        xhr: recentEntries.filter(e => e.initiatorType === 'xmlhttprequest').length
      }
    };
  })();

  result.userActivity = (function() {
    const state = window.__userActivityState || {
      lastInteraction: 0,
      interactionCount: 0,
      mouseMovements: 0,
      keyPresses: 0,
      clicks: 0,
      scrolls: 0
    };

    return {
      lastInteraction: state.lastInteraction,
      timeSinceLastInteraction: Date.now() - state.lastInteraction,
      interactionCount: state.interactionCount,
      mouseMovements: state.mouseMovements,
      keyPresses: state.keyPresses,
      clicks: state.clicks,
      scrolls: state.scrolls,
      isActive: Date.now() - state.lastInteraction < 30000
    };
  })();

  result.memoryPressure = (function() {
    if (performance.memory) {
      return {
        usedJSHeapSize: performance.memory.usedJSHeapSize,
        totalJSHeapSize: performance.memory.totalJSHeapSize,
        jsHeapSizeLimit: performance.memory.jsHeapSizeLimit,
        usageRatio: performance.memory.usedJSHeapSize / performance.memory.jsHeapSizeLimit
      };
    }
    return null;
  })();

  return result;
})();
```

### 用户活动追踪脚本

```javascript
(function() {
  if (window.__userActivityInitialized) return;
  
  window.__userActivityState = {
    lastInteraction: Date.now(),
    interactionCount: 0,
    mouseMovements: 0,
    keyPresses: 0,
    clicks: 0,
    scrolls: 0
  };

  const updateActivity = (type) => {
    const state = window.__userActivityState;
    state.lastInteraction = Date.now();
    state.interactionCount++;
    
    switch(type) {
      case 'mousemove': state.mouseMovements++; break;
      case 'keypress': state.keyPresses++; break;
      case 'click': state.clicks++; break;
      case 'scroll': state.scrolls++; break;
    }
  };

  const throttledHandlers = {
    mousemove: throttle(() => updateActivity('mousemove'), 100),
    keypress: () => updateActivity('keypress'),
    click: () => updateActivity('click'),
    scroll: throttle(() => updateActivity('scroll'), 100)
  };

  Object.entries(throttledHandlers).forEach(([event, handler]) => {
    document.addEventListener(event, handler, { passive: true });
  });

  function throttle(fn, delay) {
    let lastCall = 0;
    return function(...args) {
      const now = Date.now();
      if (now - lastCall >= delay) {
        lastCall = now;
        fn.apply(this, args);
      }
    };
  }

  window.__userActivityInitialized = true;
})();
```

## DOM 活动检测

```typescript
class DOMActivityChecker {
  private previousState: DOMState | null = null;

  async check(): Promise<DOMActivityResult> {
    const currentState = await this.captureDOMState();
    
    if (!this.previousState) {
      this.previousState = currentState;
      return { hasChanges: false, changeDetails: null };
    }

    const diff = this.compareStates(this.previousState, currentState);
    this.previousState = currentState;

    return {
      hasChanges: diff.hasChanges,
      changeDetails: diff.details
    };
  }

  private async captureDOMState(): Promise<DOMState> {
    const script = `
      (function() {
        return {
          nodeCount: document.getElementsByTagName('*').length,
          bodyInnerHTML: document.body ? document.body.innerHTML.length : 0,
          title: document.title,
          url: window.location.href,
          forms: Array.from(document.forms).map(f => ({
            id: f.id,
            name: f.name,
            inputCount: f.elements.length
          })),
          iframes: document.getElementsByTagName('iframe').length,
          scripts: document.getElementsByTagName('script').length,
          styles: document.getElementsByTagName('style').length + 
                  document.getElementsByTagName('link').length
        };
      })();
    `;

    const result = await this.cdpClient.Runtime.evaluate({
      expression: script,
      returnByValue: true
    });

    return result.result.value as DOMState;
  }

  private compareStates(prev: DOMState, curr: DOMState): {
    hasChanges: boolean;
    details: DOMChangeDetails;
  } {
    const details: DOMChangeDetails = {
      nodeCountDiff: curr.nodeCount - prev.nodeCount,
      contentLengthDiff: curr.bodyInnerHTML - prev.bodyInnerHTML,
      urlChanged: curr.url !== prev.url,
      titleChanged: curr.title !== prev.title,
      formChanges: this.compareForms(prev.forms, curr.forms),
      iframeDiff: curr.iframes - prev.iframes,
      scriptDiff: curr.scripts - prev.scripts,
      styleDiff: curr.styles - prev.styles
    };

    const hasChanges = 
      details.nodeCountDiff !== 0 ||
      details.contentLengthDiff !== 0 ||
      details.urlChanged ||
      details.titleChanged ||
      details.iframeDiff !== 0 ||
      details.scriptDiff !== 0;

    return { hasChanges, details };
  }

  private compareForms(prev: FormInfo[], curr: FormInfo[]): FormChange[] {
    const changes: FormChange[] = [];
    
    curr.forEach(form => {
      const prevForm = prev.find(f => f.id === form.id || f.name === form.name);
      if (!prevForm || prevForm.inputCount !== form.inputCount) {
        changes.push({
          formId: form.id || form.name,
          type: prevForm ? 'modified' : 'added',
          inputCountDiff: prevForm ? form.inputCount - prevForm.inputCount : form.inputCount
        });
      }
    });

    prev.forEach(form => {
      if (!curr.find(f => f.id === form.id || f.name === form.name)) {
        changes.push({
          formId: form.id || form.name,
          type: 'removed',
          inputCountDiff: -form.inputCount
        });
      }
    });

    return changes;
  }
}

interface DOMState {
  nodeCount: number;
  bodyInnerHTML: number;
  title: string;
  url: string;
  forms: FormInfo[];
  iframes: number;
  scripts: number;
  styles: number;
}

interface FormInfo {
  id: string;
  name: string;
  inputCount: number;
}

interface DOMChangeDetails {
  nodeCountDiff: number;
  contentLengthDiff: number;
  urlChanged: boolean;
  titleChanged: boolean;
  formChanges: FormChange[];
  iframeDiff: number;
  scriptDiff: number;
  styleDiff: number;
}
```

## 网络活动检测

```typescript
class NetworkActivityChecker {
  private cdpClient: CDPClient;
  private networkEnabled: boolean = false;

  async enable(): Promise<void> {
    await this.cdpClient.Network.enable();
    this.networkEnabled = true;
  }

  async check(): Promise<NetworkActivityResult> {
    const script = `
      (function() {
        const entries = performance.getEntriesByType('resource');
        const now = Date.now();
        const recentWindow = 5000;
        
        const recent = entries.filter(e => now - e.startTime < recentWindow);
        
        return {
          totalRequests: entries.length,
          recentRequests: recent.length,
          byType: {
            document: entries.filter(e => e.initiatorType === 'navigation').length,
            script: entries.filter(e => e.initiatorType === 'script').length,
            stylesheet: entries.filter(e => e.initiatorType === 'link' && e.name.includes('.css')).length,
            image: entries.filter(e => e.initiatorType === 'img').length,
            xhr: entries.filter(e => e.initiatorType === 'xmlhttprequest').length,
            fetch: entries.filter(e => e.initiatorType === 'fetch').length,
            other: entries.filter(e => !['navigation', 'script', 'link', 'img', 'xmlhttprequest', 'fetch'].includes(e.initiatorType)).length
          },
          avgDuration: recent.length > 0 
            ? recent.reduce((sum, e) => sum + e.duration, 0) / recent.length 
            : 0,
          slowRequests: recent.filter(e => e.duration > 1000).length
        };
      })();
    `;

    const result = await this.cdpClient.Runtime.evaluate({
      expression: script,
      returnByValue: true
    });

    return result.result.value as NetworkActivityResult;
  }

  async getActiveConnections(): Promise<ActiveConnection[]> {
    if (!this.networkEnabled) {
      await this.enable();
    }

    const connections: ActiveConnection[] = [];

    this.cdpClient.Network.on('requestWillBeSent', (params) => {
      connections.push({
        requestId: params.requestId,
        url: params.request.url,
        method: params.request.method,
        timestamp: params.timestamp
      });
    });

    return connections;
  }
}

interface NetworkActivityResult {
  totalRequests: number;
  recentRequests: number;
  byType: Record<string, number>;
  avgDuration: number;
  slowRequests: number;
}

interface ActiveConnection {
  requestId: string;
  url: string;
  method: string;
  timestamp: number;
}
```

## 内存压力检测

```typescript
class MemoryPressureChecker {
  private samples: MemorySample[] = [];
  private maxSamples: number = 20;
  private warningThreshold: number = 0.8;
  private criticalThreshold: number = 0.95;

  async check(): Promise<MemoryPressureResult> {
    const script = `
      (function() {
        if (!performance.memory) {
          return { available: false };
        }
        
        return {
          available: true,
          usedJSHeapSize: performance.memory.usedJSHeapSize,
          totalJSHeapSize: performance.memory.totalJSHeapSize,
          jsHeapSizeLimit: performance.memory.jsHeapSizeLimit,
          usageRatio: performance.memory.usedJSHeapSize / performance.memory.jsHeapSizeLimit
        };
      })();
    `;

    const result = await this.cdpClient.Runtime.evaluate({
      expression: script,
      returnByValue: true
    });

    const memory = result.result.value as MemoryInfo;
    
    if (!memory.available) {
      return { available: false, level: 'unknown' };
    }

    this.samples.push({
      timestamp: Date.now(),
      usageRatio: memory.usageRatio
    });

    if (this.samples.length > this.maxSamples) {
      this.samples.shift();
    }

    const trend = this.calculateTrend();
    const level = this.determineLevel(memory.usageRatio);

    return {
      available: true,
      current: memory,
      level,
      trend,
      samples: this.samples
    };
  }

  private calculateTrend(): 'increasing' | 'stable' | 'decreasing' {
    if (this.samples.length < 5) return 'stable';

    const recent = this.samples.slice(-5);
    const older = this.samples.slice(-10, -5);

    const recentAvg = recent.reduce((sum, s) => sum + s.usageRatio, 0) / recent.length;
    const olderAvg = older.length > 0 
      ? older.reduce((sum, s) => sum + s.usageRatio, 0) / older.length 
      : recentAvg;

    const diff = recentAvg - olderAvg;
    
    if (diff > 0.05) return 'increasing';
    if (diff < -0.05) return 'decreasing';
    return 'stable';
  }

  private determineLevel(usageRatio: number): 'normal' | 'warning' | 'critical' {
    if (usageRatio >= this.criticalThreshold) return 'critical';
    if (usageRatio >= this.warningThreshold) return 'warning';
    return 'normal';
  }
}

interface MemorySample {
  timestamp: number;
  usageRatio: number;
}

interface MemoryPressureResult {
  available: boolean;
  current?: MemoryInfo;
  level: 'normal' | 'warning' | 'critical' | 'unknown';
  trend: 'increasing' | 'stable' | 'decreasing';
  samples: MemorySample[];
}
```

## 内容探测决策逻辑

```typescript
class Layer2DecisionMaker {
  decide(result: ContentCheckResult): {
    mode: HeartbeatMode | 'unknown';
    confidence: number;
    needsDeeperCheck: boolean;
  } {
    if (result.userActivity?.isActive) {
      return {
        mode: 'normal',
        confidence: 0.9,
        needsDeeperCheck: false
      };
    }

    if (result.networkActivity?.recentRequests > 0) {
      return {
        mode: 'normal',
        confidence: 0.8,
        needsDeeperCheck: false
      };
    }

    if (result.domActivity?.hasChanges) {
      return {
        mode: 'normal',
        confidence: 0.7,
        needsDeeperCheck: false
      };
    }

    if (result.memoryPressure?.level === 'critical') {
      return {
        mode: 'frozen',
        confidence: 0.6,
        needsDeeperCheck: true
      };
    }

    const timeSinceActivity = result.userActivity?.timeSinceLastInteraction || 0;
    if (timeSinceActivity > 60000) {
      return {
        mode: 'idle',
        confidence: 0.7,
        needsDeeperCheck: false
      };
    }

    return {
      mode: 'unknown',
      confidence: 0.4,
      needsDeeperCheck: true
    };
  }
}
```

## 错误处理与降级

```typescript
class Layer2ErrorHandler {
  handle(error: Error, previousResult?: DetectionResult): DetectionResult {
    console.warn('[Layer2] Content check error:', error);

    if (error.message.includes('timeout')) {
      return {
        mode: 'frozen',
        confidence: 0.5,
        signals: [{
          type: 'thread_blocked',
          source: 'layer2',
          value: 'timeout',
          timestamp: Date.now(),
          weight: 0.5
        }],
        timestamp: Date.now(),
        layer: 2
      };
    }

    if (error.message.includes('target crashed')) {
      return {
        mode: 'crashed',
        confidence: 0.9,
        signals: [{
          type: 'process_frozen',
          source: 'layer2',
          value: 'target_crashed',
          timestamp: Date.now(),
          weight: 0.9
        }],
        timestamp: Date.now(),
        layer: 2
      };
    }

    return previousResult || {
      mode: 'unknown',
      confidence: 0,
      signals: [],
      timestamp: Date.now(),
      layer: 2
    };
  }
}
```
